	#选题
	1. 一份研究报告的生成的两种策略，一种是基于大纲生成内容，一种是基于内容生成大纲。
	2. 规划的 Agent 配合执行的 Agent，可以是两种方式，一种是直接生成多个任务，然后依次完成，可以支持动态调整规划的任务（相当于重写），另外一种是每次生成一个任务，可以基于执行结果动态调整下一个任务。
	3. 多 Agent 的设计，一种是无状态的，一种是有状态的，这里指的有状态指的是两个 Agent 的对话历史是有保留的，而无状态的类型就是将 Agent 当做一个工具来使用。不过我认为目前大多数 Agent 都只会用到无状态的类型，仅仅有少部分环节的 Agent 是需要维持状态，比如规划和执行的 Agent 应该是有状态的，因为他们是一个有责任关系的组合，而其他功能性的 Agent 往往只需要保持无状态的即可。不过核心还是在复杂性和成本方面上进行的考虑。
	4. 工具的无状态和有状态之分，类似 notebook 的代码执行工具就是一个典型的有状态的工具，我觉得狭义上来讲，notebook 工具执行，还有接入的数据库类型的工具都算有状态的工具。广义上来讲。
