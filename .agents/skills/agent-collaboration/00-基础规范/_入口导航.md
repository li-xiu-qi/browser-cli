## 系统环境

**重要提示**: 本知识库运行于 **Windows + PowerShell** 环境。

| 项目 | 说明 |
|------|------|
| **操作系统** | Windows 10/11 |
| **终端环境** | PowerShell 5.1+ |
| **Shell 工具** | 使用 PowerShell cmdlet，而非 Bash |

### 命令使用规范

**必须使用 PowerShell 语法**:
- 文件列表: `Get-ChildItem` 或 `dir` / `ls`（别名）
- 创建目录: `New-Item -ItemType Directory` 或 `mkdir`
- 移动文件: `Move-Item` 或 `mv`
- 复制文件: `Copy-Item` 或 `cp`
- 删除文件: `Remove-Item` 或 `rm`
- 读取内容: `Get-Content` 或 `cat`
- 路径分隔符: 使用 `\` 或 `/`（PowerShell 兼容两者）

**避免使用**:
- `head`, `tail`, `grep`, `awk`, `sed` 等 Unix 工具
- `&&`, `||` 链式操作（PowerShell 使用 `;` 或 `-and`/`-or`）
- 反斜杠转义（使用 PowerShell 引号规则）

## 规范优先级（重要！）

当规范冲突时，按以下优先级执行：

| 优先级 | 规范来源 | 说明 |
|--------|----------|------|
| 1 (最高) | **项目 README.md** | Projects/xxx/README.md 中的"项目特定规范" |
| 2 | **项目级 Skill** | `Projects/xxx/.agents/skills/` 中的 Skill |
| 3 | **Areas/My_Persona/** | 个人偏好设定 |
| 4 | **AGENTS.md** | 全局 AI 协作规范 |
| 5 | **全局 Skill** | `.agents/skills/` 中的 Skill |
| 6 | **Skill 默认行为** | Skill 内部默认规则 |

**AI 执行任务前必须**：
1. 检查工作目录是否有 README.md
2. 优先阅读其中的"项目特定规范"部分
3. 再查阅全局规范

## AI 工作规范（教训沉淀）

### 使用技能前必须先读 SKILL.md

**血淋淋的教训**：在绘制 Graphviz 图表时，因未先阅读 `text-visualization` 的场景1文档，导致所有 SVG 出现 viewBox 偏移问题，需要全部返工。

**强制规范**：
1. **执行任何涉及技能的任务前，先阅读对应 SKILL.md**
2. 不要凭经验想当然，技能文档中可能有坑和最佳实践
3. 生成产物后立即验证（如 SVG 检查 viewBox 是否为 0,0）

**检查清单**：
- [ ] Graphviz → 阅读 `text-visualization` (场景1)
- [ ] 案例收集 → 阅读 `case-study-collector/SKILL.md`
- [ ] 排版规范 → 阅读 `obsidian-markdown-formatter/SKILL.md`
- [ ] 其他任何技能 → 先阅读对应 SKILL.md 再动手
