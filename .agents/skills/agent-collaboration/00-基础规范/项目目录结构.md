# 项目目录结构

**定位**: 知识库与 AI 配置的双层目录体系  
**作用**: 帮助 AI 快速定位文件位置，减少目录搜索

---

## 第一层：笔记内容层（PARA）

```
笔记专用/
├── 0_Inbox/              # 收集箱 → 定期清理
├── Projects/             # 项目 → 完成后归档
│   └── YYYY-MM-项目名/
├── Areas/                # 领域 → 长期维护
│   ├── AI协作学/
│   ├── My_Persona/
│   ├── Scripts/
│   ├── 认知思考/
│   ├── 科学决策/
│   └── ...
├── Resources/            # 资源 → 参考资料
│   ├── 课程资料/
│   ├── 书籍/
│   ├── 附件/
│   └── 参考项目/         # 开源项目本地镜像 → 学习参考用
├── Archives/             # 归档 → 已完成
│   └── YYYY/
├── cliper_datas/         # 抓取文章 → 按主题分类
└── 日记/                 # 日常记录
    └── YYYY-MM-DD.md
```

### 分类速查

| 目录 | 用途 | 示例 |
|------|------|------|
| 0_Inbox | 临时存放，等待分类 | 未整理的速记、待处理链接 |
| Projects | 有截止时间的短期任务 | 2026-02-案例库建设 |
| Areas | 长期维护的责任领域 | 认知思考、科学决策 |
| Resources | 可长期参考的资料 | 课程笔记、书籍摘录 |
| 参考项目 | 开源项目本地镜像 | PAI、Skills 等他人优秀项目 |
| Archives | 已完结的内容 | 2025年的已完成项目 |
| cliper_datas | 网络抓取的文章 | 按主题组织的剪藏内容 |

---

## 第二层：AI 配置层（.agents）

```
.agents/
├── skills/               # 技能库
│   ├── agent-collaboration/      # AI 协作规范 ← 本目录
│   ├── knowledge-base-manager/
│   ├── obsidian-markdown-formatter/
│   ├── resource-crawler/
│   └── ...
├── tools/                # 工具脚本
└── workflows/            # 工作流定义
    ├── course-manuscript-refinement/
├── knowledge-processing/
    └── ...
```

### 核心 Skill 速查

| Skill | 用途 |
|-------|------|
| agent-collaboration | AI 协作规范体系（本目录） |
| knowledge-base-manager | 知识库结构管理 |
| obsidian-markdown-formatter | Markdown 排版规范检查 |
| resource-crawler | 资源抓取（网页+音视频） |
| knowledge-indexing | 索引文件命名规范 |
| learning-notes | 学习笔记（课程笔记+案例收集） |
