# 排版样例

本目录包含不同排版层级的示例文件，供参考学习。

## 样例列表

| 文件 | 层级 | 说明 |
|------|------|------|
| example-001-极简版.md | 一级 | 纯标题层级，无任何装饰 |
| example-002-轻度装饰.md | 二级 | 核心概念引用、关键数字加粗 |
| example-003-结构化.md | 三级 | 四级标题、步骤加粗、完整结构 |
| example-004-反例_过度装饰.md | - | 展示过度使用格式的问题 |

## 使用建议

1. **默认使用** `example-002-轻度装饰.md` 的格式
2. **快速记录** 参考 `example-001-极简版.md`
3. **复杂教程** 参考 `example-003-结构化.md`
4. **检查自己** 对比 `example-004-反例_过度装饰.md` 避免同样错误
