# 知识管理课程案例集

**说明**: 本目录收录 Truman 在一堂直播课中分享的具体笔记案例，展示不同段位的知识管理实践。

---

## 设计理念

本案例集采用"索引+案例文档"的双层结构设计：

- **索引文件**（本文件）：提供分类导航、快速选择、核心原则
- **案例文件**：详细记录单个笔记的背景、思路、内容、洞察

这种设计降低选择成本，便于建立案例之间的联系。

## 核心原则

1. **形式不重要**: 可以只记一个书名、一句话、一张图
2. **主题学习 > 碎片化学习**: 做一次全量研究胜过多次零散阅读
3. **笔记=能力**: 刻意练习笔记要与能力训练合一
4. **深度可练习**: 强迫自己往深想四到五层
5. **索引+案例**: 用双层结构组织案例库
