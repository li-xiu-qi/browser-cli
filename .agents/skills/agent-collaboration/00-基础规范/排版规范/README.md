# 排版规范

本目录包含 Obsidian Markdown 笔记的排版规范，确保格式简洁、清晰、易读。

## 核心原则

**克制优先**
- 格式服务于内容，不喧宾夺主
- 能不用则不用，必须用才用
- 少即是多

## 快速参考

**三级格式层级**:

| 层级 | 特点 | 适用场景 |
|------|------|----------|
| 极简 | 仅标题，无装饰 | 快速记录、草稿 |
| 轻度装饰 | 核心概念引用、关键数字加粗 | 知识笔记、案例整理 |
| 结构化 | 四级标题、步骤加粗 | 教程、操作指南 |
