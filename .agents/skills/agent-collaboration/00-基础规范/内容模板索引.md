## 使用规范

### 模板使用原则

1. **先选模板，再动手** - 明确使用哪个模板后再开始写作
2. **保持结构一致** - 同一类内容使用相同模板
3. **灵活调整** - 模板是指导，不是枷锁，可根据实际需要删减模块
4. **链接关联** - 使用 wikilink 建立模板内容与其他笔记的关联

### 模板更新流程

发现模板需要改进时：
1. 在 [[changelogs/README|变更日志]] 中记录
2. 更新源 SKILL.md 中的模板定义
3. 同步更新本索引（如结构有变化）
