# 浏览器自动化规范

本规范定义项目中所有浏览器自动化相关的标准和工具使用方法。

## 核心原则

**统一浏览器数据目录**: 所有需要浏览器自动化的工具 **必须** 使用统一的浏览器数据目录，实现登录状态共享。

```
.agents/browser_user_data/
```

## 为什么统一目录？

1. **登录一次，多处使用** - 登录状态被所有工具共享
2. **环境一致性** - 避免多个浏览器实例的冲突
3. **持久化** - 关闭浏览器后登录状态不会丢失
4. **用户体验** - 不需要重复登录同一个网站

## 工具位置

### 通用浏览器登录工具

**路径**: `.agents/tools/browser-login/`

**用途**: 通用的浏览器登录工具，支持任意网站登录

**使用方式**:
```powershell
# 登录任意网站
.venv\Scripts\python.exe .agents\tools\browser-login\login.py <URL> --name <site_name>

# 示例：Z-Library 登录
.venv\Scripts\python.exe .agents\tools\browser-login\login.py https://zh.zlib.li --name zlibrary

# 查看已保存的 session
.venv\Scripts\python.exe .agents\tools\browser-login\login.py --list
```

### Z-Library 工具

**路径**: `.agents/skills/resource-crawler/tools/zlibrary/`

**用途**: Z-Library 搜索下载（已包装了通用登录）

**使用方式**:
```powershell
# 登录
uv run python .agents/skills/resource-crawler/tools/zlibrary/login.py

# 搜索下载
uv run python .agents/skills/resource-crawler/tools/zlibrary/zlib_api.py auto "书名"
```

## 开发规范

### 新建浏览器自动化工具

如果你的 Skill 需要浏览器自动化，**必须**遵循以下规范：

#### 1. 使用统一的用户数据目录

```python
from pathlib import Path

# 所有工具必须使用这个目录
user_data_dir = Path(".agents/browser_user_data")
```

#### 2. Playwright 启动配置

```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch_persistent_context(
        user_data_dir=str(user_data_dir),
        headless=False,  # 或 True，根据需求
        args=['--disable-blink-features=AutomationControlled']
    )
    # ... 自动化操作
```

#### 3. 保存登录状态

使用通用登录工具保存 cookies：

```python
# 保存 cookies 到 browser-login/config/
cookies = context.cookies()
with open(f'config/{site_name}_cookies.json', 'w') as f:
    json.dump(cookies, f)
```

#### 4. 读取登录状态

```python
# 从 browser-login 读取已保存的 cookies
import json
from pathlib import Path

cookies_file = Path(".agents/skills/resource-crawler/tools/zlibrary/config/zlibrary_cookies.json")
if cookies_file.exists():
    with open(cookies_file, 'r') as f:
        cookies = json.load(f)
```

## 配置目录

| 用途 | 路径 | 说明 |
|------|------|------|
| 浏览器数据 | `.agents/browser_user_data/` | 所有工具共享的浏览器数据 |
| 登录配置 | `.agents/tools/browser-login/config/` | 保存的 cookies |
| Z-Library 配置 | `.agents/skills/resource-crawler/tools/zlibrary/config/` | Z-Library 专用 token |

## 安全注意事项

1. **敏感信息** - `browser_user_data/` 包含登录凭证，**不要**提交到 Git
2. **已添加到 .gitignore**:
   ```
   .agents/browser_user_data/
   .agents/**/browser_user_data/
   ```
3. **定期清理** - 旧的无用 session 可以手动删除

## 故障排查

### 浏览器启动失败

```powershell
# 检查 Playwright 是否安装
.venv\Scripts\python.exe -m playwright install chromium
```

### 登录状态丢失

```powershell
# 重新登录
.venv\Scripts\python.exe .agents\tools\browser-login\login.py <URL> --name <site>
```

### 检查浏览器数据目录

```powershell
# 确认目录存在
ls .agents/browser_user_data/
```
