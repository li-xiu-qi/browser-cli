# 2026-02-17: 新增规范索引质量管理指南

**类型**: 规范新增

## 背景

规范索引（如 `_入口导航.md`、`SKILL.md`）是 AI 和用户查找规范的核心入口。在实践中发现索引存在完整性、准确性、一致性、可用性等方面的问题，需要系统化的质量管理方法论。

## 改进

1. 新建 `01-开发指南/规范索引质量管理.md`
   - 定义索引质量的四个维度（完整性、准确性、一致性、可用性）
   - 提供四维度检测方法
   - 索引质量问题分级（致命/严重/中等/轻微）
   - 具体修复方法（添加遗漏、修复链接、统一格式、优化层级）
   - 索引维护机制（定期审查、变更同步、质量标准清单）

2. 更新相关导航
   - `_入口导航.md`：添加规范索引质量管理入口
   - `SKILL.md`：添加功能描述
   - `规范体系管理.md`：更新目录清单和关联文档
