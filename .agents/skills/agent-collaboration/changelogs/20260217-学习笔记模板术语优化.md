# 2026-02-17 学习笔记模板术语优化

**变更类型**: 模板术语优化  
**影响范围**: `learning-notes` Skill  
**变更原因**: 避免术语过度使用，让笔记表达更自然

---

## 变更内容

### 修改前
模板中统一使用 **"关键洞察"** 作为章节标题

### 修改后
模板中改用 **"划重点"** 作为章节标题

---

## 变更理由

在实际使用中发现，"关键洞察"这个表述过于正式且重复率高，导致：

1. **机械感强** - 像教科书分点，缺少个人笔记的自然感
2. **使用过度** - 模板里多次出现，使用者容易套用过多
3. **不够口语化** - 人在做笔记时更常说"这里要划重点"而不是"这是我的关键洞察"

"划重点"更符合：
- 中文笔记的自然表达习惯
- 学习时的真实心理状态（"这里要划下来"）
- 避免模板痕迹过重的问题

## 使用建议

对于已存在的笔记：
- **不强制修改** - 已有笔记保持原样，避免批量改动
- **新笔记使用** - 从本变更日期后创建的新笔记采用"划重点"
- **灵活调整** - 也可以根据内容特点使用其他表述，如：
  - "核心洞见"
  - "一句话总结"
  - "我的理解"
  - "举个例子"

模板是指导，不是枷锁，根据内容自然选择最合适的表述。
