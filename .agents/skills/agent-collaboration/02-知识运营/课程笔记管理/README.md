# 课程笔记管理

本目录包含课程学习笔记的规范化管理指南，涵盖从课程收集、笔记整理到知识萃取的全流程。

## 适用场景

- 一堂课程、训练营
- 在线网课（得到、混沌、极客时间等）
- 线下培训、工作坊
- 会议演讲、TED Talk

## 快速开始

**标准课程笔记流程**:

```
收集课程信息 → 记录/转录内容 → 结构化整理 → 知识萃取 → 行动转化
```

**文件命名规范**:
```
{课程名称}-{讲师}-{日期}.md
例如: 五层决策模型-老喻-20240215.md
```
