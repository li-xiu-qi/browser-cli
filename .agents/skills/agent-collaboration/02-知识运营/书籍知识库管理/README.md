# 10 - 书籍知识库管理

## 定位

本目录包含**书籍知识库（AI协作版）**的完整管理规范和工作流程。

区别于传统的"阅读书单"，本知识库追求**"能被AI引用"**而非"我读完没有"。

## 核心理念

```
传统阅读管理          AI协作知识库
─────────────────    ─────────────────
读没读、读到哪页  →   AI能不能准确引用
待读/阅读中/已读  →   raw/indexed/chunked/ref
完成阅读目标      →   可被AI检索引用
```

## 3步入库流程（简版）

```
Step 1: 放入 Library/
       ↓
Step 2: 4维度评分（40秒）
       - text_quality: 文本质量
       - structure: 结构清晰度
       - density: 知识密度
       - ai_ready: AI适配度
       ↓
Step 3: 按等级处理
       - A级(4.5+): chunked（已分块，AI即插即用）
       - B级(3.5-4.5): indexed（已索引，AI可引用）
       - C级(<3.5): raw（仅存档）
```
