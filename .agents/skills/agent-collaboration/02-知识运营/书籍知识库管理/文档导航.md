## 工具链速查

### 可用工具

| 工具 | 路径 | 功能 | 输入 → 输出 |
|------|------|------|-------------|
| **epub2obsidian** | `.agents/tools/epub2obsidian/` | EPUB转Markdown | .epub → 高质量Markdown |
| **ocr-to-obsidian** | `.agents/tools/ocr-to-obsidian/` | OCR识别 | .pdf/图片 → Markdown |
| **wechat-book-to-pdf** | `.agents/tools/wechat-book-to-pdf/` | 微信读书抓取 | URL → A4 PDF |

### 工具命令速查

**EPUB → Markdown**
```bash
node .agents/tools/epub2obsidian/index.js "Library/xxx.epub" --category "02"
```

**微信读书 → PDF**
```bash
# 登录（首次）
node .agents/tools/wechat-book-to-pdf/index.js login

# 抓取（后台）
node .agents/tools/wechat-book-to-pdf/index.js crawl "URL" -bg
```

**PDF → Markdown（OCR）**
```bash
python .agents/tools/ocr-to-obsidian/main.py \
  "Library/xxx.pdf" \
  --output "Areas/书籍/00_待分类/xxx.md" \
  --background
```

## 质量评分体系

### 4维度评分（1-5分）

| 维度 | 评估内容 | 基于 |
|------|----------|------|
| **text_quality** | 文本质量 | Markdown内容：错字？乱码？ |
| **structure** | 结构清晰度 | Markdown标题层级 |
| **density** | 知识密度 | 内容类型：干货vs水分 |
| **ai_ready** | AI适配度 | 综合判断 |

### 质量等级

| 等级 | 综合分 | 状态 | 处理 |
|------|--------|------|------|
| A | 4.5-5.0 | chunked | AI可直接引用 |
| B | 3.5-4.4 | indexed | 可全文检索 |
| C | 2.5-3.4 | raw | 用时再处理 |
| D | < 2.5 | ref | 保留不依赖 |

## 常用操作清单

### 入库一本 EPUB（5分钟）

```bash
# 1. 放入Library
mv xxx.epub Areas/书籍/Library/

# 2. 转换
node .agents/tools/epub2obsidian/index.js \
  "Areas/书籍/Library/xxx.epub" \
  --category "02"

# 3. 评估质量，更新卡片 frontmatter
# 4. 视情况 chunked 或 indexed
```

### 入库微信读书（30-120分钟）

```bash
# 1. 抓取
node .agents/tools/wechat-book-to-pdf/index.js crawl "URL" -bg

# 2. 等待完成，复制到Library

# 3. 决定是否OCR
python .agents/tools/ocr-to-obsidian/main.py \
  "Library/xxx.pdf" \
  --output "Areas/书籍/05_分类/xxx.md"

# 4. 校对后 indexed
```

### 批量评估现有书籍

1. 使用 [批量评分工作表.md](./批量评分工作表.md)
2. 抽样检查各分类
3. 标记A级书籍优先处理

## 一句话总结

> **EPUB直接转，微信先抓PDF，扫描看质量，图册不折腾。高质chunked入库，低质ref备用。**
