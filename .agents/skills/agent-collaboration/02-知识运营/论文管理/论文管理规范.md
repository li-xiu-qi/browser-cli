## 目录结构

```
resources/论文阅读/
├── README.md                          # 主入口：快速开始指南
├── _INDEX-论文阅读.md                  # 论文索引：快速查阅所有论文
└── papers/                            # 论文文件夹集合
    └── {关键词}_{标题英文}/             # 每篇论文一个文件夹
        ├── meta.json                  # 论文元数据
        ├── original.pdf               # 原始英文PDF
        ├── translated-zh.pdf          # 中文翻译版（可选）
        ├── translated-dual.pdf        # 中英对照版（可选）
        ├── ocr.md                     # OCR提取的Markdown（可选）
        ├── images/                    # OCR提取的图片（可选）
        └── notes.md                   # 阅读笔记（可选）
```

## 论文文件夹规范

### 文件夹命名

```
{缩写或关键词}_{标题英文}
```

**命名原则**:
- 使用英文，用下划线 `_` 连接
- 关键词首字母大写（CamelCase）
- 标题过长时可适当缩写

**示例**：
- `LoRA_Low-Rank_Adaptation_of_Large_Language_Models`
- `Attention_Is_All_You_Need`
- `Transformer_Architecture`
- `LinearAttention_Transformers_are_RNNs_Fast_Autoregressive_Transformers_with_Linear_Attention`

### 文件说明

| 文件 | 必需 | 说明 |
|------|:----:|------|
| `meta.json` | ✅ | 论文元数据（标题、作者、年份、会议、关键词等） |
| `original.pdf` | ✅ | 原始英文论文 |
| `ocr.md` | ❌ | OCR提取的文本内容 |
| `translated-zh.pdf` | ❌ | 中文翻译版 |
| `translated-dual.pdf` | ❌ | 中英对照版 |
| `images/` | ❌ | 提取的图片素材 |
| `notes.md` | ❌ | **个人阅读笔记** |

## 元数据格式 (meta.json)

### 完整示例

```json
{
  "title": "LoRA: Low-Rank Adaptation of Large Language Models",
  "authors": ["Edward J. Hu", "Yelong Shen", "Phillip Wallis", "Zeyuan Allen-Zhu", "Yuanzhi Li", "Shean Wang", "Lu Wang", "Weizhu Chen"],
  "year": 2021,
  "venue": "ICLR 2022",
  "keywords": ["PEFT", "Fine-tuning", "LLM", "Low-rank"],
  "tags": ["#LLM", "#PEFT"],
  "url": "https://arxiv.org/abs/2106.09685",
  "doi": null,
  "added_date": "2026-02-16",
  "translation_status": "未翻译",
  "ocr_status": "已完成",
  "reading_status": "未读",
  "rating": null,
  "notes": ""
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| `title` | string | ✅ | 论文标题 |
| `authors` | array | ✅ | 作者列表 |
| `year` | number | ✅ | 发表年份 |
| `venue` | string | ✅ | 发表会议/期刊 |
| `keywords` | array | ❌ | 关键词（技术术语） |
| `tags` | array | ❌ | 分类标签（见下方标签表） |
| `url` | string | ❌ | 论文链接（arXiv/DOI） |
| `doi` | string/null | ❌ | DOI（如有） |
| `added_date` | string | ✅ | 添加日期（YYYY-MM-DD） |
| `translation_status` | string | ❌ | 翻译状态：未翻译/已翻译 |
| `ocr_status` | string | ❌ | OCR状态：未提取/已完成 |
| `reading_status` | string | ❌ | 阅读状态：未读/进行中/已读 |
| `rating` | number/null | ❌ | 评分（1-5星） |
| `notes` | string | ❌ | 简短备注 |

## 新增论文完整流程

### 步骤 1: 创建论文文件夹

```powershell
$paperName = "关键词_论文标题英文"
New-Item -ItemType Directory -Path "resources/论文阅读/papers/$paperName"
```

### 步骤 2: 放入原始 PDF

将下载的 PDF 重命名为 `original.pdf` 放入文件夹。

### 步骤 3: 创建 meta.json

填写论文的基本信息，参考上方格式。

### 步骤 4: [可选] OCR 提取文本

```bash
# 方法1：使用 content-converter Skill 的 OCR 工具
uv run python .agents/skills/content-converter/tools/ocr/ocr.py \
  "papers/{论文标题}/original.pdf" \
  -o "papers/{论文标题}/" \
  -c "论文" \
  -t "关键词1,关键词2"
```

```python
# 方法2：使用 pymupdf 快速提取（推荐简单场景）
import fitz
doc = fitz.open("original.pdf")
text = ""
for page in doc:
    text += page.get_text()
doc.close()
with open("ocr.md", "w", encoding="utf-8") as f:
    f.write(text)
```

### 步骤 5: [可选] 翻译成中文

```bash
# 使用 PDF 翻译工具
uv run python .agents/skills/content-converter/tools/pdf-translator/pdf-translator.py \
  "papers/{论文标题}/original.pdf"
```

输出文件：
- `original-zh.pdf` → 重命名为 `translated-zh.pdf`
- `original-dual.pdf` → 重命名为 `translated-dual.pdf`

### 步骤 6: [可选] 创建阅读笔记

创建 `notes.md` 记录阅读心得。

### 步骤 7: 在索引中登记

在 `resources/论文阅读/_INDEX-论文阅读.md` 中添加论文条目。

## 阅读笔记存放位置

### 方式一：论文文件夹内（推荐）

- **位置**: `papers/{论文标题}/notes.md`
- **优点**: 笔记和论文在一起，方便对照
- **适合**: 单篇论文的深度阅读笔记
- **模板**:

```markdown
# 阅读笔记：论文标题

**作者**: 
**发表**: 
**标签**: 

## 核心贡献

## 关键方法

## 实验结果

## 个人思考

## 待深入研究
```

### 方式二：Areas 领域下

- **位置**: `Areas/阅读方法/` 或新建主题领域
- **优点**: 同主题论文聚合，方便知识串联
- **适合**: 跨论文的主题研究、综述笔记
- **示例**:
  - `Areas/阅读方法/注意力机制研究.md`
  - `Areas/阅读方法/线性注意力论文笔记.md`

## 阅读笔记写作规范

### 公式书写规范

**必须使用 LaTeX 格式书写公式**，确保在 Obsidian 中正确渲染。

#### 行内公式

使用单个 `$` 包裹：
```markdown
复杂度从 $O(n^2)$ 降到 $O(n)$
特征映射函数 $\phi(x) = \text{elu}(x) + 1$
```

#### 块级公式

使用双 `$$` 包裹：
```markdown
$$
\text{Attention} = \text{softmax}\left(\frac{QK^T}{\sqrt{d}}\right) V
$$
```

#### 常见符号规范

| 符号 | LaTeX | 示例 |
|------|-------|------|
| 上标 | `^` | `$n^2$` → $n^2$ |
| 下标 | `_` | `$x_i$` → $x_i$ |
| 分数 | `\frac{}{}` | `$\frac{a}{b}$` → $\frac{a}{b}$ |
| 开方 | `\sqrt{}` | `$\sqrt{d}$` → $\sqrt{d}$ |
| 求和 | `\sum` | `$\sum_{i=1}^n$` → $\sum_{i=1}^n$ |
| 点乘 | `\cdot` | `$Q \cdot K$` → $Q \cdot K$ |
| 转置 | `^T` | `$K^T$` → $K^T$ |
| 函数 | `\text{}` | `$\text{softmax}$` → $\text{softmax}$ |
| 希腊字母 | `\phi, \sigma` | `$\phi(x)$` → $\phi(x)$ |

#### 公式书写示例

```markdown
**传统 Softmax 注意力**:
$$
\text{Attention} = \text{softmax}\left(\frac{QK^T}{\sqrt{d}}\right) V \quad \text{(复杂度 } O(n^2)\text{)}
$$

**线性注意力**:
$$
\text{Attention} = \phi(Q) \cdot (\phi(K)^T \cdot V) \quad \text{(复杂度 } O(n)\text{)}
$$

其中 $\phi(x) = \text{elu}(x) + 1$ 是特征映射函数。
```

### 笔记内容建议

1. **一句话总结**: 放在开头，提炼核心贡献
2. **核心概念**: 解释论文解决什么问题
3. **数学原理**: 关键公式 + 直观解释
4. **实验结果**: 关键数据 + 性能对比
5. **个人思考**: 优点、局限、启发
6. **延伸阅读**: 相关论文推荐
