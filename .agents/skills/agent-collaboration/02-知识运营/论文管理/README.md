## 实际存储位置

论文实际存储在：`resources/论文阅读/`

```
resources/论文阅读/
├── README.md                 # 论文阅读管理说明
├── _INDEX-论文阅读.md         # 论文索引
└── papers/                   # 论文文件夹集合
    └── {论文标题}/
        ├── meta.json         # 元数据
        ├── original.pdf      # 原始论文
        ├── ocr.md            # OCR文本（可选）
        └── notes.md          # 阅读笔记（可选）
```
