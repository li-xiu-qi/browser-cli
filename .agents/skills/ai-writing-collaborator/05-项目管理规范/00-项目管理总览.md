# 写作项目管理规范

**定位**: AI 协作写作的项目管理方法论  
**配套**: 与 [[../SKILL]] 协同使用

---

## 核心原则

### 1. 元数据驱动
- 不因为状态变化而移动文件
- 用元数据（标签、索引）标记文章状态
- 原文始终保留在 `03_工作区/`

### 2. 流程标准化
- 统一的工作区结构（捕获→提取→输出）
- 统一的复盘流程
- 统一的风格 Skill 迭代流程

### 3. 渐进式完善
- 先完成再完美
- 复盘驱动改进
- Skill 随文章积累而进化

---

## 目录结构标准

```
Projects/写作/
├── AGENTS.md                      # 项目级 AI 协作规范 ⭐
├── README.md                      # 项目说明（给人类看）
├── 01_选题思考区/                  # 灵感收集
├── 02_选题库/                      # 选题储备
├── 03_工作区/                      # 写作工作区（核心）
│   └── [文章标题]/
│       ├── 捕获.md                # 输入：灵感收集
│       ├── 提取.md                # 加工：逻辑大纲
│       ├── 输出.md                # 产出：最终成稿
│       ├── 参考文献/               # PDF、论文等
│       └── assets/                # 配图、附件
└── 04_Skill素材库/                 # 复盘与风格管理
    ├── _INDEX.md                  # 文章状态索引
    └── 复盘记录/                   # 复盘文档
```

---

## 关键文件：AGENTS.md vs README

| 文件 | 读者 | 用途 | 内容 |
|------|------|------|------|
| **AGENTS.md** | AI | 执行规范 | 必须做什么、禁止做什么、执行步骤 |
| **README.md** | 人类 | 项目说明 | 是什么、为什么、怎么用、快速开始 |

**执行逻辑**:
1. AI 执行任务前，先读 **AGENTS.md**（约束和步骤）
2. 如有疑问，参考 **README.md**（背景和说明）
3. 核心能力来自 **全局 Skill**（本 Skill）

## AGENTS.md 模板

每个写作项目应有自己的 `AGENTS.md`：

```markdown
# [项目名] AI 协作规范

**执行者**: AI  
**配套 Skill**: [[.agents/skills/ai-writing-collaborator/]]

## 执行前必读
1. 先读全局 Skill
2. 再读本规范

## 核心约束
- [ ] 必须...
- [ ] 禁止...

## 目录规范（强制执行）
...

## 写作协作流程
...

## 项目专属规则（覆盖全局）
- 署名：xxx
- 风格偏好：...

## 关联文档
- [[全局 Skill]]
- [[本项目管理规范]]
```

## 与全局 Skill 的关系

```
全局 Skill（本 Skill）
    ├── 写作方法论（风格创建、迭代、场景协作）
    ├── 管理规范（目录结构、生命周期、复盘流程）
    └── 模板（技术博客、公众号等）
            ↓ 被项目 AGENTS.md 引用/覆盖
项目 AGENTS.md
    ├── 引用全局规则
    ├── 覆盖/扩展规则
    └── 定义项目专属约束
            ↓ 指导 AI 执行
实际写作任务
```

**优先级**（冲突时）：
1. 项目 AGENTS.md（最高）
2. 全局 Skill
3. 通用最佳实践

## 关键文件说明

| 文件 | 用途 | 必须遵循的规范 |
|------|------|----------------|
| `捕获.md` | 灵感收集 | 只管存，不管理；允许混乱 |
| `提取.md` | 逻辑加工 | 结构优先；完成思考后再写作 |
| `输出.md` | 最终成稿 | 读者视角；遵循风格 Skill |
| `_INDEX.md` | 状态索引 | 及时更新；正例/反例分类 |
| `复盘记录` | 风格分析 | 按模板；提炼可复用规则 |
| `AGENTS.md` | AI 规范 | 执行前必读；约束 AI 行为 |
| `README.md` | 项目说明 | 人类了解项目；AI 备用参考 |

## AI 协作流程

### 新文章流程

```
1. 用户说"我要写一篇关于 XXX 的文章"
    ↓
2. AI 检查/创建 03_工作区/[文章]/ 目录结构
    ↓
3. AI 加载风格 Skill（如用户指定）
    ↓
4. 按 捕获→提取→输出 流程协作
    ↓
5. 用户说"已发布"
    ↓
6. AI 更新 04_Skill素材库/_INDEX.md
    ↓
7. 提醒用户复盘
```

### 复盘流程

1. 读取原文（`03_工作区/[文章]/输出.md`）
2. 按 [[03-复盘流程规范|复盘模板]] 分析
3. 确定分类（正例 ⭐ / 反例 ❌）
4. 更新 `_INDEX.md` 状态
5. 根据复盘更新风格 Skill

详见 [[04-与Skill协作流程]]
