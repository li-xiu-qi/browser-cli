## 视觉理解评测集速查

### 9个评测维度

```
1. 目标对比 (Object Comparison)     - 细粒度目标识别
2. 图片复刻 (Image Replication)      - 网页/UI复刻能力
3. 图片排序 (Image Sequencing)       - 事件逻辑推理
4. 空间逻辑 (Spatial Logic)          - 空间关系推理
5. 空间变换 (Spatial Transformation) - 视觉错觉理解
6. OCR识别 (Text Recognition)        - 文字识别能力
7. 表格识别 (Table Recognition)      - 结构化数据提取
8. 成语与时钟 (Cultural & Time)      - 文化/时间理解
9. 文本推理 (Textual Reasoning)      - 逻辑推理能力
```

### 难度分布

- ⭐⭐ 基础: 6个case
- ⭐⭐⭐ 进阶: 10个case
- ⭐⭐⭐⭐ 困难: 7个case
- ⭐⭐⭐⭐⭐ 极限: 2个case

## 使用场景

### 场景1: 写模型评测文章

参考 [[视觉理解评测集/评测集说明\|评测集说明]] 中的case，进行模型测试：

```markdown
## 评测方法
本次评测使用刘聪NLP整理的《视觉理解评测集》，包含9个维度25个case...

### 目标对比能力
测试模型对细粒度目标的识别能力...

[测试结果]
```

### 场景2: 快速测试模型某方面能力

直接选取对应维度的case进行测试：

```bash
# 测试空间推理能力
选取：Case 4.1 (鸡腿走迷宫)、Case 4.2 (六面体展开图)

# 测试OCR极限能力
选取：Case 6.3 (究极OCR)
```

### 场景3: 对比不同模型

使用相同case测试不同模型，制作对比表格：

| case | 模型A | 模型B | 模型C |
|------|-------|-------|-------|
| 不开心的冰块 | ✅ | ❌ | ✅ |
| 迷宫复刻 | ⚠️ | ✅ | ❌ |
| ... | ... | ... | ... |

## 新增资源流程

1. **收集case**: 从优质评测文章中提取测试case
2. **分类整理**: 按能力维度分类，标注难度
3. **编写说明**: 创建评测集说明文档（参考模板）
4. **准备assets**: 将测试图片放入对应assets目录
5. **更新索引**: 在本文件添加条目
