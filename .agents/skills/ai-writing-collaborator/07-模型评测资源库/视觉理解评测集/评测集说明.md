# 视觉理解评测集（VLM Benchmark）

**来源**: 刘聪NLP《千问3.5开源，原生多模态架构冲击AGI，附一手全面实测！》  
**整理日期**: 2026-02-16  
**适用场景**: 多模态大模型（VLM）能力评测

---

## 评测集概述

本评测集包含 **9个维度、25个测试case**，覆盖视觉理解模型的核心能力：

| 维度 | case数 | 测试能力 |
|------|--------|----------|
| 目标对比 | 3 | 细粒度目标识别 |
| 图片复刻 | 2 | 网页/UI复刻能力 |
| 图片排序 | 2 | 事件逻辑推理 |
| 空间逻辑 | 2 | 空间关系推理 |
| 空间变换 | 3 | 视觉错觉、变换理解 |
| OCR识别 | 3 | 文字识别（手写/极小字） |
| 表格识别 | 2 | 结构化数据提取 |
| 成语与时钟 | 3 | 文化理解、时间识别 |
| 文本推理 | 3 | 逻辑推理、常识推理 |

## 评测维度详解

### 1. 目标对比（Object Comparison）

测试模型对细粒度目标的识别能力。

**Case 1.1: 奔跑的人**
- **Prompt**: "找到图片中奔跑的人，并返回行列序号，比如：几行几列。"
- **难度**: ⭐⭐
- **预期**: 正确识别动态目标
- **结果**: ✅ 正确

![奔跑的人](assets/01-01-奔跑的人.jpg)

**Case 1.2: 磨损的眼镜**
- **Prompt**: "找到磨损的眼镜，并返回行列序号，比如：几行几列。"
- **难度**: ⭐⭐⭐
- **预期**: 识别细微特征（磨损）
- **结果**: ✅ 正确

![磨损的眼镜](assets/01-02-磨损的眼镜.png)

**Case 1.3: 不开心的冰块** ⭐高难度
- **Prompt**: "找到不开心的冰块，并返回行列序号，比如：几行几列。"
- **难度**: ⭐⭐⭐⭐
- **预期**: 理解情绪+物理状态（冰块不会开心）
- **结果**: ❌ **错误** - "这个确实有难度了"

![不开心的冰块](assets/01-03-不开心的冰块.jpg)

## 使用指南

### 如何使用本评测集

1. **选择维度**: 根据要测试的能力选择对应维度
2. **准备图片**: 使用 `assets/` 目录下的对应图片
3. **执行测试**: 使用上述Prompt进行测试
4. **记录结果**: 对比"预期"和"实际结果"
5. **综合评价**: 参考"难度"和"评价"进行能力评估

### 评测标准建议

| 评级 | 正确率 | 说明 |
|------|--------|------|
| ⭐⭐⭐⭐⭐ | >90% | 优秀，可商用 |
| ⭐⭐⭐⭐ | 70-90% | 良好，大部分场景可用 |
| ⭐⭐⭐ | 50-70% | 一般，需要人工校验 |
| ⭐⭐ | <50% | 较差，仅作参考 |

### 难度分级说明

- ⭐⭐: 基础能力，主流模型应该都能完成
- ⭐⭐⭐: 进阶能力，部分模型可能失败
- ⭐⭐⭐⭐: 高难度，只有顶级模型能完成
- ⭐⭐⭐⭐⭐: 极限挑战，当前模型普遍失败
