# 场景3：访谈整理

**场景**: 将播客/访谈录音转为文章  
**典型用例**: 播客 show notes、访谈稿整理、会议纪要

---

## 为什么访谈需要双 Skill

### 访谈整理的两层工作

1. **内容层**: 提取信息、整理结构、去口语化
2. **风格层**: 转化为你的文章风格

### 双 Skill 协作

| Skill | 职责 |
|-------|------|
| 访谈整理 Skill | 提取要点、理清逻辑、去废话 |
| 写作风格 Skill | 转化为你的表达方式 |

---

## 工作流程

```
录音/转录稿 → 访谈 Skill 处理 → 风格 Skill 重写 → 手动修改 → 更新 Skill
```

---

## 详细步骤

### 步骤1：获取转录稿

来源：
- 播客平台的自动转录
- 飞书妙记、讯飞听见等工具
- 手动转录

### 步骤2：访谈 Skill 初处理

**提示词**：
```markdown
请对以下访谈转录稿进行初步整理：

整理要求：
1. 提取核心观点（去掉闲聊、重复、口癖）
2. 按主题重新组织段落
3. 补充必要的上下文（如提到的作品、人物需说明）
4. 保留有价值的金句和案例
5. 标注需要核实的信息

【转录稿】
[粘贴]
```

### 步骤3：风格 Skill 重写

**提示词**：
```markdown
请将以下整理后的访谈内容，改写成我的文章风格。

【我的写作风格 Skill】
[粘贴]

【整理后的内容】
[粘贴]

要求：
1. 严格遵循我的风格
2. 以我的口吻输出（第一人称）
3. 保留所有核心观点和金句
4. 可以适当添加我的点评和背景说明
```

### 步骤4：手动修改

**重点检查**：
- [ ] 第一人称口吻是否自然
- [ ] 转述的观点是否准确
- [ ] 你的点评是否符合你的风格
- [ ] 整体是否读起来像你自己写的

---

## 特殊场景

### 场景A：多人访谈

**处理策略**：

```markdown
整理要求补充：
- 清晰标注每个观点的发言人
- 保留对话感，不要改成纯叙述
- 突出观点冲突和碰撞
- 最后添加我的总体点评
```

### 场景B：我作为受访者

**处理策略**：

如果访谈中的"我"是受访者：
- 直接用原文整理
- 检查是否有口误需要修正
- 补充当时没表达清楚的部分

### 场景C：会议纪要

**简化流程**：

```markdown
请将以下会议录音转录稿整理成会议纪要：

要求：
1. 提取决议事项（TODO）
2. 提取关键讨论点
3. 按我的风格重写，但要简洁
4. 保留必要的背景信息
```

## 维护访谈专用 Skill

```markdown
# 筱可-访谈整理风格

## 基础风格
- 继承：[[筱可-个人风格-V3.0]]

## 访谈特有规则
- 第一人称转述他人观点时，明确标注来源
- 金句用引用格式：> "金句内容" ——发言人
- 我的点评用单独段落，前面加"我的看法："
- 复杂观点用列表呈现

## 结构模板
1. 一句话总结访谈主题
2. 核心观点（3-5个）
3. 详细展开
4. 金句摘录
5. 我的点评
```

## 示例

### 转录稿片段

> 主持人：您刚才提到 AI 会改变写作，具体是指什么？
> 嘉宾：呃...就是说，我觉得吧，AI 不是替代写作，而是改变写作的方式。你看，以前我们要花很多时间在资料收集上，现在 AI 可以帮我们做这部分。但是创意、观点，这些还是人来做的。
> 主持人：那您觉得作家会失业吗？
> 嘉宾：不会不会，至少短期内不会。我觉得啊，AI 是工具，就像以前的打字机、电脑一样，是让人写得更好的工具。

### 整理后

> **AI 如何改变写作**
>
> 嘉宾认为，AI 不会替代写作，而是改变写作方式。过去作者需要花大量时间收集资料，这部分工作现在可以交给 AI。但创意和观点的产生，仍然是人类的核心能力。
>
> 关于"作家是否会失业"的问题，嘉宾明确表示短期内不会。他将 AI 比作打字机和电脑——是让人写得更好的工具，而非替代者。

### 风格化后

> **AI 是打字机，不是对手**
>
> 访谈中聊到 AI 对写作的影响，嘉宾的观点我很认同：
>
> AI 不会替代写作，但会改变写作方式。
>
> 以前我们得花大量时间泡图书馆、查资料，这部分苦力活现在可以丢给 AI。但真正的创意、独特的观点，还是得人来。
>
> 有人担心作家会失业。嘉宾的答案是：不会，至少短期内不会。
>
> 他把 AI 比作打字机、电脑——这些工具没有让作家失业，反而让写作更高效。AI 也一样。
>
> **我的看法**：这个"打字机"的比喻很精准。与其担心被替代，不如思考怎么用好这个新工具。
