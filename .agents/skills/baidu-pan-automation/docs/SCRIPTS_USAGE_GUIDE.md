## 🚀 快速开始

### 第一步：配置 Token（必须）

```bash
# 进入项目目录
cd Projects/2026-02-百度网盘自动化集成

# 运行授权脚本（会自动打开浏览器）
python scripts/get_access_token.py

# 按提示操作：
# 1. 登录百度账号（如果未登录）
# 2. 点击"同意授权"
# 3. 脚本会自动提取并保存 Token
```

**Token 会保存到三个地方**:
- `.env` - 环境变量（脚本读取）
- `.agents/mcp.json` - MCP 配置
- `.token_info.json` - Token 详情

### 4. 执行删除/移动操作

```bash
python scripts/correct_api_execute.py
```

**⚠️ 警告**: 这会实际修改网盘内容！

**执行前请确认**:
1. 已运行 `verify_token.py` 确认 Token 有效
2. 已运行 `deep_compare_all.py` 确认操作清单
3. 重要数据已备份

**脚本会执行**:
- 删除完全重复的文件夹
- 移动独有的内容到目标目录

### 场景 4: 课程字幕获取与整理

**功能**: 从百度网盘下载课程视频的字幕，并转换为 Markdown 文稿

**配置**: 编辑 `scripts/get_subtitles.py`，设置以下变量：
```python
ACCESS_TOKEN = "your_token_here"  # 从 .env 或 mcp.json 获取
BASE_DIR = "/我的资源/课程目录"     # 网盘中的课程路径
```

**执行步骤**:
```bash
# 1. 获取字幕（下载 SRT 文件到 subtitles/ 目录）
python scripts/get_subtitles.py

# 输出示例:
# ======================================================
# 黄执中《亲密关系中的语言控制》课程字幕获取工具
# ======================================================
# 🔍 正在搜索课程视频...
# 找到 12 个视频文件:
# [1/12] 📹 [1]--引发罪恶感(22)情绪卸责.mp4
# ✅ 字幕已保存: subtitles/[1]--引发罪恶感(22)情绪卸责.srt

# 2. 转换为 Markdown 文稿
python scripts/convert_srt_to_md.py

# 输出:
# transcripts/[1]--引发罪恶感(22)情绪卸责.md
# ...
```

**输出文件**:
- `subtitles/` - SRT 格式字幕文件（原始）
- `transcripts/` - Markdown 格式文稿（智能分段，适合阅读）

**注意事项**:
- 需要百度网盘 SVIP 会员才能获取字幕
- 字幕生成依赖百度网盘 AI，可能需要等待
- 转换后的 Markdown 已智能分段，每段约 300 字

## ⚠️ 安全注意事项

### Token 安全
- Token 有效期 30 天，过期需重新运行 `get_access_token.py`
- 不要将 `.env` 和 `.token_info.json` 提交到 Git
- Token 已配置在 `.gitignore` 中

### 操作安全
- **删除操作**会进入回收站，10天内可恢复
- **移动操作**不可逆，操作前请确认路径
- 建议先运行分析脚本，确认操作清单后再执行

### API 限制
- 频率限制: 100次/分钟
- 批量操作: 每次最多操作 100 个文件
- 大文件夹需要分批处理

## 🐛 故障排查

### Token 过期
```
错误: errno = -6 或 31034
解决: python scripts/get_access_token.py
```

### API 调用失败
```
错误: 请求失败
检查:
1. python scripts/verify_token.py
2. 检查网络连接
3. 确认路径存在
```

```bash

pip install requests playwright
```
