Exp 7: Neural Network Training
Objective
Train a 3-layer neural network on MNIST dataset. Compare different optimization algorithms (SGD, Adam, RMSprop).
Parameters
Parameter Value Notes
Input size 784 28×28 pixels
Hidden layers 2 256, 128 neurons
Output classes 10 Digits 0-9
Learning rate 0.001 Adam optimizer
Batch size 64 Mini-batch
Epochs 50 Early stopping at 45
Results
Optimizer Final Loss Test Accuracy Training Time
SGD 0.245 92.3% 12.5 min
Adam 0.089 97.8% 10.2 min
RMSprop 0.112 96.5% 11.1 min
Observations
Adam converged fastest with lowest loss. SGD showed slower convergence but good generalization. No overfitting observed with dropout rate 0.5.
