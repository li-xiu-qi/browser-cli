# K2A 技能萃取工作流

从深度笔记中提取 Agent 可执行的技能（Prompts）。

## 一句话说明

**只增不减**地从优质笔记中萃取 AI 可执行的技能，建立"理论"与"工具"的双向连接。

## 适用场景

- 已有写得很好的课程笔记
- 笔记中包含可复用的"术"（如写作技巧、思维模型、调试步骤）
- 想将方法论转化为 AI 工具

## 核心原则

1. **只增不减**：严禁修改或删减原始笔记中的任何知识细节
2. **按需执行**：只有包含明确"方法论"、"操作步骤"时才提取

## 处理流程

```
笔记分析 → [技能识别] → [Prompt构建] → [双向锚定] → 技能资产
```

## 快速启动

1. **提供笔记文件路径**
2. **说"使用 K2A 技能萃取工作流"**
3. AI 判断是否可萃取，如可则生成 Prompt 并建立链接

1. **原始笔记**：保持原汁原味，内容完整
2. **Prompt 文件**：（如果有）独立的可执行指令文件
