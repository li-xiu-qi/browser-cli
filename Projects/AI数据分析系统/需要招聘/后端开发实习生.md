
## 职位描述

作为后端开发实习生，你将参与核心业务的开发和维护工作。你将使用 Python 生态工具构建高性能的服务，并参与系统架构的优化。

## 岗位职责

- 开发后端微服务。主要使用 Python 和 FastAPI 框架。
    
- 负责数据库的设计和维护。会使用 Supabase 或者 PostgreSQL 进行数据存储。
    
- 使用 Docker 管理开发和部署环境。
    
- 处理异步任务和缓存优化。提高系统响应速度。
    

## 职位要求

- 熟悉 Python 语言。能够使用 FastAPI 构建稳定的后端服务。
    
- 了解 Supabase 或者 PostgreSQL。清楚常用的数据库概念和操作方式。
    
- 具备数据库设计经验。能够独立完成业务表的结构设计。
    
- 会用 Redis。有处理缓存的实际经验。
    
- 熟悉 Docker。会使用 Docker Compose 部署服务。
    
- 掌握 Git 常用命令。能够进行团队协作开发。
    
- 了解 Celery。会构建和管理任务队列。
    

## 个人素质

- 学习能力强。对新技术有好奇心，能快速上手新工具。
    
- 沟通主动。会积极汇报工作进度，遇到问题能及时写报告反馈。
