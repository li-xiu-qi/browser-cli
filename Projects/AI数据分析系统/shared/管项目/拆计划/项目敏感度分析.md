## 六维敏感度评估

| 敏感度类型 | 评估 | 说明 | 管理重点 |
|------------|------|------|----------|
| **预算敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 是否有严格预算上限？超支是否致命？ | 成本控制、采购审批 |
| **时间敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 是否有对外承诺的 deadline？延期影响？ | 关键路径、进度监控 |
| **人力敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 人手是否固定？能否临时加人？ | 任务拆解、backup 机制 |
| **协作敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 是否涉及跨部门？是否需要建立信任？ | 角色分工、同步机制 |
| **风险敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 是否第一次做？失败过？技术难度？ | 预案准备、buffer 预留 |
| **老板敏感** | ⬜ 高 / ⬜ 中 / ⬜ 低 | 是否是 A 高度重视的战略项目？ | 汇报频率、关键节点 |
