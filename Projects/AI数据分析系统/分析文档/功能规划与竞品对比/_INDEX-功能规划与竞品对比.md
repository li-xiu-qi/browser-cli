## 核心结论速览

### 竞品定位对比

```
DB-GPT          → 企业级数据应用框架 (DAG 编排)
CAMEL           → 多 Agent 研究框架 (角色扮演)
smolagents      → 轻量 Agent 库 (简洁透明)
Claude SDK      → Claude 专用 SDK (异步会话)
AIASys          → 工业数据分析平台 (Host-Worker)
```

### AIASys 差异化优势

1. **工业场景聚焦** - 轧机、电机等设备专用分析
2. **三级上下文压缩** - Micro/Auto/Manual 分层
3. **Host-Worker 分工** - 对话与执行分离
4. **Jupyter 深度集成** - 状态持久化、Notebook 导出

### 关键借鉴方向

| 来源 | 借鉴内容 | 应用场景 |
|------|----------|----------|
| DB-GPT | DSL + DAG 可视化 | 复杂分析流程展示 |
| CAMEL | Agent 工厂 + Workforce | 多 Agent 扩展 |
| Claude SDK | MCP + Hooks | 生态接入 + 拦截器 |

## 录屏调研工具

为了更直观地了解竞品产品，提供了自动化录屏调研工具：

### 功能特点
-  自动按顺序打开 14 个竞品网站
-  引导使用 Windows 自带 Xbox Game Bar 录屏
-  包含国内外主流工业数据分析平台
-  录屏文件自动整理建议

### 使用步骤
1. 确保已启用 Xbox Game Bar（设置 > 游戏 > Xbox Game Bar）
2. 双击 `启动录屏调研.bat` 或右键运行 `.ps1` 脚本
3. 按提示浏览网站，使用 `Win + Alt + R` 开始/停止录屏
4. 录屏完成后将视频发给我，我可以帮你分析产品功能

### 录屏文件命名建议
```
01-容知日新-智能运维平台.mp4
02-寄云-NeuSeer-预测性维护.mp4
03-华为-FusionPlant-工业互联网.mp4
...
```
