**创建日期**: 2026-02-08

##  准备工作

### 1. 启用 Xbox Game Bar

Windows 自带录屏功能，需要确保已启用：

1. 按 `Win + I` 打开设置
2. 选择 **游戏** → **Xbox Game Bar**
3. 打开开关：**使用 Xbox Game Bar 录制游戏剪辑、屏幕截图和广播**
4. 记住录屏快捷键：**Win + Alt + R**

### 2. 检查 Chrome 浏览器

确保已安装 Google Chrome 浏览器。

##  使用方法

### 方式一：右键运行（推荐）

1. 找到文件 `竞品网站录屏调研.ps1`
2. **右键** → **使用 PowerShell 运行**
3. 如果提示"执行策略"问题，请看下方的【常见问题】

### 方式二：PowerShell 命令行

1. 按 `Win + X`，选择 **终端(管理员)** 或 **PowerShell**
2. 切换到脚本所在目录：
   ```powershell
   cd "C:\Users\你的用户名\Documents\projects\obsidian_projects\笔记专用\Projects\AI数据分析系统\分析文档\功能规划与竞品对比"
   ```
3. 执行脚本：
   ```powershell
   .\竞品网站录屏调研.ps1
   ```

##  录屏操作步骤

每个网站的浏览流程：

1. **脚本自动打开网站**（Chrome 新标签页）
2. **你开始录屏**：按 `Win + Alt + R`
3. **浏览网站**：
   - 滚动页面查看功能介绍
   - 点击产品菜单查看详情
   - 查看解决方案页面
   - 如有演示视频也可以播放
4. **停止录屏**：再按 `Win + Alt + R`
5. **继续下一个**：在 PowerShell 窗口按 `Enter`

##  录屏文件管理

### 默认保存位置
```
C:\Users\你的用户名\Videos\Captures\
```

### 文件命名建议
Xbox Game Bar 默认命名格式：`应用名 日期 时间.mp4`

建议你手动重命名为：
```
01-容知日新-智能运维平台.mp4
02-寄云-NeuSeer-预测性维护.mp4
03-华为-FusionPlant-工业互联网.mp4
...
```

### 整理目录
脚本会创建文件夹：
```
C:\Users\你的用户名\Videos\竞品调研录屏\
```

建议将整理好的录屏文件放入此目录，方便后续分析。

## ⚠️ 常见问题

### Q1: 执行策略错误
**错误提示**：`无法加载文件，因为在此系统上禁止运行脚本`

**解决方法**（以管理员身份运行 PowerShell）：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
输入 `Y` 确认，然后重新运行脚本。

### Q2: Xbox Game Bar 无法录屏
**可能原因**：
- 桌面或文件资源管理器不支持 Game Bar 录屏
- 需要打开一个应用（如 Chrome）才能录屏

**解决方法**：
- 确保 Chrome 已打开后再开始录屏
- 或使用其他录屏软件：OBS Studio、ShareX 等

### Q3: Chrome 无法打开
**检查**：
- Chrome 是否已安装
- 尝试将 `chrome.exe` 改为 `msedge`（使用 Edge 浏览器）

### Q4: 某些网站无法访问
**说明**：
- 部分国外网站（如 Seeq、C3.ai）可能需要科学上网
- 如果打不开，可以手动搜索该产品的国内介绍页面
