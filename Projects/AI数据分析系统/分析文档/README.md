## 重要洞察速览

### 五大架构模式

```
Pipeline模式: 用户输入 → 理解 → 生成 → 执行 → 输出
              (premsql, PandasAI)

Agent模式: 用户输入 → Agent → [观察→思考→行动]循环 → 输出
           (Vanna 2.0, TaskWeaver, DB-GPT)

RAG模式: 用户输入 → 语义检索 → 组装上下文 → LLM生成 → 输出
          (Vanna, SQLBot, WrenAI)

语义层模式: 用户输入 → 语义解析 → 语义模型 → SQL生成 → 输出
            (WrenAI, SuperSonic)

人机协作模式: 用户 ←──→ AI推荐/UI编辑 ←──→ 输出
              (Data Formulator, PyGWalker)
```

### 技术栈趋势

| 层级 | 主流选择 | 使用率 |
|------|---------|--------|
| 后端语言 | Python | 95% |
| Web框架 | FastAPI | 70% |
| LLM框架 | LangChain | 60% |
| 前端框架 | React | 80% |
| 可视化 | Vega-Lite | 60% |
