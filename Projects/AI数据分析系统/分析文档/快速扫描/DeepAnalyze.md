## 2. 核心能力速览
- 主要功能:
  - 端到端数据科学管道（数据准备、分析、建模、可视化、报告生成）
  - 开放式数据研究（支持结构化/半结构化/非结构化数据）
  - 自动生成分析师级研究报告
  - 基于 DeepSeek-R1-0528-Qwen3-8B 微调
  - 500K 训练数据开源
- 技术亮点:
  - **端到端自治**：无需人工干预完成完整数据分析任务
  - **全栈开源**：模型、代码、数据全部开放
  - **课程式训练**：单能力微调 -> 多能力冷启动 -> RL 强化学习

## 3. 架构概览
DeepAnalyze 是一个**端到端训练的专用模型**，而非框架：

```
User Input + Data Files
    |
    v
DeepAnalyze-8B (VLLM 部署)
    |
    v
Code Execution Environment
    |
    v
Analysis Report / Visualization
```

- **Agent类型**: 端到端模型（基于 ReAct/Chain-of-Thought 训练）
- **工具注册**: 内置代码执行能力，通过生成 Python/SQL 代码与环境交互
- **记忆管理**: 多轮对话中的上下文保持

### 4.1 值得借鉴的设计
- **课程式训练策略**：Single-ability -> Multi-ability -> RL 的训练流程
- **端到端自治**：展示了大模型在数据分析任务上的潜力上限
- **全栈开源模式**：模型、数据、代码全开放的生态建设

### 4.2 可复用的代码/模块
- `deepanalyze/ms-swift/` - 模型训练框架（基于 ms-swift）
- `deepanalyze/SkyRL/` - RL 训练框架
- `scripts/` - 训练脚本（single.sh、multi_coldstart.sh、multi_rl.sh）
- `API/` - OpenAI 风格 API 服务
