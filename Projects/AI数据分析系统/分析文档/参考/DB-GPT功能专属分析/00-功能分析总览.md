## 二、功能模块清单

### 2.1 展示层功能

| 模块 | 核心功能 | 目标用户 | 价值定位 |
|------|----------|----------|----------|
| **探索广场** | 应用发现、应用搜索、应用收藏、应用发布 | 所有用户 | 降低发现成本，快速开始使用 |
| **在线对话** | 多场景对话、历史会话、参数调节、文件上传 | 终端用户 | 统一交互入口，数据智能对话 |
| **应用管理** | 应用创建、AWEL编排、Agent配置、发布管理 | 开发者/管理员 | 低代码构建自定义应用 |
| **模型评测** | 评测数据集、评测任务、效果对比、报告导出 | 算法工程师/管理员 | 量化效果，辅助选型优化 |

### 2.2 场景层功能

| 场景 | 输入 | 输出 | 解决的问题 |
|------|------|------|------------|
| **Chat Knowledge** | 文档/问题 | 答案+出处 | 企业文档检索效率低 |
| **Chat DB** | 自然语言 | SQL+结果+图表 | 业务人员不会写SQL |
| **Chat Excel** | Excel文件+问题 | 分析结果+图表 | Excel分析门槛高 |
| **Chat Dashboard** | 报表+追问 | 深入分析 | 静态报表无法交互 |
| **Chat Agent** | 复杂任务描述 | 任务执行结果 | 需要多步骤协作的场景 |

### 2.3 编排层功能

| 功能 | 能力描述 | 使用场景 |
|------|----------|----------|
| **AWEL Flow** | 拖拽式工作流编排 | 构建固定流程的AI应用 |
| **Prompt 管理** | 提示词模板管理 | 统一提示词风格，复用优秀模板 |
| **Agent 开发** | 多智能体协作配置 | 复杂决策、需要分工的场景 |

## 三、功能使用路径

### 路径一：业务人员 - 开箱即用
```
探索广场 → 选择应用 → 在线对话 → 获得结果
    │
    ├─ 推荐应用：热门场景快速入口
    ├─ 全部应用：浏览所有可用应用
    └─ 我的收藏：常用应用快捷访问
```

### 路径二：数据分析师 - 深度分析
```
在线对话 → 选择场景 → 上传数据 → 对话分析 → 导出结果
    │
    ├─ Chat DB：查询数据库
    ├─ Chat Excel：分析Excel
    └─ Chat Dashboard：交互式报表
```

### 路径三：应用开发者 - 构建应用
```
应用管理 → 创建应用 → AWEL编排 → 测试调试 → 发布到广场
    │
    ├─ 选择模板：基于现有模板创建
    ├─ 从零编排：拖拽节点构建流程
    └─ Agent配置：设置多智能体协作
```

### 路径四：算法工程师 - 优化效果
```
模型评测 → 上传数据集 → 创建评测 → 对比结果 → 优化配置
    │
    ├─ Text2SQL评测：优化SQL生成效果
    ├─ RAG评测：优化知识检索效果
    └─ 模型对比：选择最佳模型
```
