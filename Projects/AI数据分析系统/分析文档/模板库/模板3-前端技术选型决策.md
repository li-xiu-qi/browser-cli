## 七、技术规范

### 7.1 代码规范

```
项目结构:
├── src/
│   ├── components/     # 通用组件
│   ├── views/          # 页面组件
│   ├── hooks/          # 自定义 Hooks
│   ├── stores/         # 状态管理
│   ├── utils/          # 工具函数
│   └── styles/         # 样式文件
├── public/
└── docs/
```

### 7.2 命名规范

| 类型 | 命名规则 | 示例 |
|------|---------|------|
| **组件** | PascalCase | `ChartViewer.tsx` |
| **Hook** | camelCase + use | `useVegaLite.ts` |
| **工具函数** | camelCase | `formatData.ts` |
| **常量** | UPPER_SNAKE_CASE | `MAX_PAGE_SIZE` |
| **样式文件** | camelCase.module.css | `chartViewer.module.css` |
