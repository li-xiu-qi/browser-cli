## 三、架构设计

### 3.1 数据流架构

```
原始数据
    ↓
数据转换层
    ├── 过滤
    ├── 聚合
    ├── 排序
    └── 计算字段
    ↓
图表配置生成
    ├── 映射字段到视觉通道
    ├── 设置刻度/范围
    └── 配置交互
    ↓
渲染引擎
    ├── 布局计算
    ├── 图形绘制
    └── 动画处理
    ↓
交互层
    ├── 事件监听
    ├── 状态更新
    └── 重渲染
```

### 3.2 图表配置结构

```typescript
// 图表配置接口（以 Vega-Lite 为例）
interface ChartConfig {
  // 数据
  data: {
    values?: DataRecord[];
    url?: string;
    name?: string;
  };
  
  // 标记
  mark: {
    type: 'bar' | 'line' | 'point' | 'area' | 'rect' | 'arc';
    tooltip?: boolean;
    // ...其他属性
  };
  
  // 编码
  encoding: {
    x?: EncodingChannel;
    y?: EncodingChannel;
    color?: EncodingChannel;
    size?: EncodingChannel;
    shape?: EncodingChannel;
    tooltip?: EncodingChannel;
  };
  
  // 变换
  transform?: Transform[];
  
  // 配置
  config?: ChartConfigStyle;
}

interface EncodingChannel {
  field: string;
  type: 'quantitative' | 'ordinal' | 'nominal' | 'temporal';
  aggregate?: 'count' | 'sum' | 'mean' | 'median' | 'min' | 'max';
  bin?: boolean | BinConfig;
  scale?: ScaleConfig;
  axis?: AxisConfig;
  legend?: LegendConfig;
}
```
