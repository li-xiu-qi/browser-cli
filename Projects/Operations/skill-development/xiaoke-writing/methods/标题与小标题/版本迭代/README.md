# 标题与小标题版本迭代

这里不放当前生效规则，只放准备试的新方案。

## 什么时候写进来

- 要整体重写标题判断逻辑
- 要同时改主标题、标题选择和小标题口径
- 想保留 `v1 / v2` 方案对照，再决定是否吸收进现行规则

## 命名约定

- `v1-标题与小标题方案.md`
- `v2-标题与小标题方案.md`

## 每版最少写清

- 这版想解决什么问题
- 这版改了哪些判断标准
- 这版影响哪些现行文件
- 这版现在是试验稿，还是已经准备回收

## 当前入口

- 现行入口仍然在 [../README.md](../README.md)
- 不要把这里的试验稿直接当成默认规则
