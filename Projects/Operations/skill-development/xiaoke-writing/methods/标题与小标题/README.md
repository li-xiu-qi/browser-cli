# 标题与小标题

这里收当前默认生效的主标题方法、小标题方法和版本迭代入口。

## 先看什么

- 还在生标题候选，先看 [公众号标题方法论](./公众号标题方法论.md)
- 已经有多个标题版本要比较，先看 [公众号标题选择规范](./公众号标题选择规范.md)
- 原始材料是对谈、播客、演讲重编排稿，先看 [思考类文章小标题方法](./思考类文章小标题方法.md)
- 要复盘标题，回看 [标题复盘模板](../../复盘模板/标题复盘模板.md)

## 当前结构

| 文件或目录 | 作用 |
|---|---|
| [公众号标题方法论.md](./公众号标题方法论.md) | 从正文价值反推主标题 |
| [公众号标题选择规范.md](./公众号标题选择规范.md) | 多个标题版本之间怎么取舍 |
| [思考类文章小标题方法.md](./思考类文章小标题方法.md) | 思考类、对谈重编排稿的小标题推进方法 |
| [版本迭代/](./版本迭代/README.md) | 标题与小标题大改版时的 `v1 / v2` 方案区 |

## 版本迭代怎么用

- 小修小补，直接改现行文件
- 如果这次不是补几句，而是准备重写一整套标题判断，先写到 [版本迭代/](./版本迭代/README.md)
- 新方案稳定后，再把结论回收进现行文件，不让活动入口越来越散
