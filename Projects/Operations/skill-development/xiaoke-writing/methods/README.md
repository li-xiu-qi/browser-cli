# 方法模块

这里收当前仍在用的、独立的**方法模块**。

## 与 `current/` 和 `versions/` 的分工

- `current/`：写稿的**总入口**，指向各个子模块
- `methods/`：单个方法模块的**当前稳定版本**
- `versions/`：**完整规范主线**的版本历史

## 当前模块

| 模块 | 路径 | 说明 |
|------|------|------|
| 标题与小标题 | [./标题与小标题/](./标题与小标题/) | 主标题、小标题的生成与选择方法 |

## 维护原则

- 单个方法模块的稳定版本放在这里
- 如果某个方法模块有大改，会在 `versions/` 中生成新版本线
- 新方法稳定后，`current/README.md` 会补充入口
