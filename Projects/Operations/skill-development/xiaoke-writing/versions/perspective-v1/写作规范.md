# 写作规范-我的视角-v1

## 定位

这份文档只解决一件事：

- 当一篇文章的价值，一部分来自作者真实的判断变化时，怎么把“我的视角”写对

它不是通用写作总规范。

它更像主线规范旁边的一份专题补充，适合私号思考文、判断型文章、对谈重编排稿、长访谈整理稿。

如果你要的是最完整的主线，先回 [v9-写作规范.md](./v9-写作规范.md)。

## 什么时候优先用“我的视角”

下面这些情况，默认优先考虑这条写法：

1. 文章价值的一部分，来自作者真实认知变化
2. 这篇不是纯整理，而是“我为什么开始这样想”
3. 材料本身已经有人能概述，读者更想知道“你为什么信这个判断”
4. 私号写作需要补信任感，不适合退回无主体的中性总评

常见场景：

- 思考类文章
- 判断型文章
- 对谈、播客、长访谈重编排稿
- 方法论长文

## 什么时候不要硬用

下面这些情况，不要为了显得更真诚，硬把全文改成第一人称：

1. 教程稿
2. 配置排错文
3. 纯操作说明
4. 资源清单
5. 对象本身已经足够具体，不需要靠作者判断来建立信任

一句话说：

- “我的视角”适合补信任感
- 不适合替代信息本身

## 核心原则

“我的视角”不是多写几个“我”。

它真正要解决的是两件事：

1. 让读者知道你的判断从哪里来
2. 让读者知道这篇文章为什么不是无主体整理稿

所以更稳的做法不是：

- 我觉得
- 我觉得
- 我觉得

更稳的做法是：

1. 我原来怎么想
2. 这次什么材料、什么场景、什么经历改了我
3. 这个变化具体落在哪个对象和判断上
4. 我把这个收获还给读者

## 更稳的起手骨架

### 写法一：原判断 -> 新判断

- 我原来会怎么想
- 这次为什么不再这么想
- 我现在更相信什么

适合：

- 对谈重编排稿
- 观点文
- 方法论判断文

### 写法二：旧经验不够 -> 新经验补上

- 我以前习惯怎么做
- 这次为什么发现那套不够
- 新的判断具体补了什么

适合：

- 工作流类文章
- AI 协作类文章
- 经验复盘稿

### 写法三：我被哪一句停住了

- 我最先停在哪一句
- 这句话为什么会让我改判断
- 它具体提醒了我什么

适合：

- 播客、访谈、演讲整理稿
- 引用驱动的思考文

## 容易写坏的几种方式

### 1. 站在材料外面做总评

这类句子很常见：

- 这场对谈最有价值的地方是
- 难得就在这里
- 把这些判断摊开讲了一遍
- 下面这几个判断值得带走

这些句子的问题不是绝对错误。

问题在于，它们会把作者放在材料外面，读者会知道你在总结，但不知道你为什么相信这套总结。

### 2. 把“我的视角”写成情绪流水账

“我的视角”不等于：

- 可以满篇感受词
- 可以一路“我觉得”
- 可以靠口语化制造亲近感

更稳的要求是：

- 说话位置是真人
- 判断对象是具体的
- 句子里有真实变化
- 但表达仍然克制、清楚、有边界

### 3. 只换成第一人称，句子还是空

这是最容易误判的一种。

句子里虽然有了“我”，但如果后面接的还是：

- 一个漂亮的反转句
- 后背发紧
- 很有意思
- 很有冲击力

那问题并没有真正解决。

更稳的写法还是要落回：

- 我具体看到了什么
- 我为什么开始警觉
- 这个变化具体落在哪个对象上

## 句子怎么改才更像真的在分享判断

例如：

- 不稳：`这场对谈最有价值的地方，是把这些判断讲得很透`
- 更稳：`这次听完这场对谈，我第一次比较清楚地意识到，主动式 AI 和我之前熟悉的那种对话式 AI，其实不是同一类东西`

- 不稳：`这是一个漂亮的反转句`
- 更稳：`这句话很反常识，但它提醒我，工具一旦进了系统，人也会跟着变`

- 不稳：`最吸引人的，是他对这件事的定义`
- 更稳：`我更在意的，是他把这件事重新讲清楚了`

## 和怪词的关系

“我的视角”这条线特别容易和怪词绑在一起。

原因很简单：

- 作者一旦开始写第一人称
- 就很容易想让句子更像文章
- 然后顺手补进一些有气氛、但不够具体的词

所以这条线要额外记一条：

- 第一人称不是怪词豁免权

如果一句话里的力量主要来自词味，不来自对象、动作和关系，那就算有了“我”，这句话还是不稳。

## 最后检查

- [ ] 这篇文章的价值，是否真的一部分来自作者判断变化
- [ ] 开头是否清楚交代了“我原来怎么想”和“这次哪里被改了”
- [ ] “我”后面是否很快落到具体对象和具体判断
- [ ] 全文是否让读者知道你为什么相信这套判断
- [ ] 有没有重新滑回“最有价值的地方是”“难得就在这里”这类无主体总评
- [ ] 有没有把“我的视角”写成情绪流水账
- [ ] 有没有靠怪词或感受词去替代真正的判断

## 一句话收口

“我的视角”不是为了把文章写得更像感想。

它真正的作用，是把作者的判断来源交代清楚，让文章从“无主体总评”变成“我为什么会这样判断”。
