# 当前默认规则

这里收当前默认生效的写稿规则和检查清单，不收版本化演化稿。

## 先看什么

- 要写稿或改稿，先看 [筱可公众号写作规范](./筱可公众号写作规范.md)
- 要写贴图或多图文，先看 [贴图写作规范](./贴图写作规范/贴图写作规范-v1.md)
- 要定标题、比标题或改小标题，先看 [标题与小标题](../methods/标题与小标题/README.md)
- 要把定稿送进公众号草稿箱，先看 [../../wechat-writing/current/Wenyan服务器发布SOP.md](../../wechat-writing/current/Wenyan服务器发布SOP.md)
- 要补表达和检查，默认先看 [写作规范-我的视角-v2](../versions/perspective-v2/写作规范.md)，再看 [用词偏好](./用词偏好.md)、[敏感内容检查清单](./敏感内容检查清单.md)、[思考类文章生产SOP](./思考类文章生产SOP.md)、[v9-写作规范](../versions/v9/写作规范.md)

## 当前结构

| 文件或目录 | 作用 |
|---|---|
| [筱可公众号写作规范.md](./筱可公众号写作规范.md) | 当前默认写稿总入口 |
| [思考类文章生产SOP.md](./思考类文章生产SOP.md) | 思考类、方法论、系统分析长文的生产补充 |
| [用词偏好.md](./用词偏好.md) | 项目级表达偏好 |
| [写作规范-我的视角-v2.md](../versions/perspective-v2/写作规范.md) | 当前优先使用的专题补充线，适合私号思考文、判断型文章、对谈重编排稿，以及有真实认知变化的对比或汇总文章 |
| [v9-写作规范.md](../versions/v9/写作规范.md) | 当前最细的长文版主线，在对象定义权基础上继续补“怪词”这层句子自然度问题 |
| [敏感内容检查清单.md](./敏感内容检查清单.md) | 发布前敏感内容检查 |
| [内容形态优先级-基于对标研究.md](./内容形态优先级-基于对标研究.md) | 对标研究沉淀出的内容形态优先级 |
| [贴图写作规范/](./贴图写作规范/) | 公众号图片消息的写作规范 |

## 维护边界

- 当前有效规则留在这里
- 标题与小标题统一放在 [../methods/标题与小标题/](../methods/标题与小标题/README.md)
- 历史版本和专题补充线统一放在 [../versions/](../versions/README.md)
- 公众号发布规则不再放这里，统一看 [../../wechat-writing/current/Wenyan服务器发布SOP.md](../../wechat-writing/current/Wenyan服务器发布SOP.md)
