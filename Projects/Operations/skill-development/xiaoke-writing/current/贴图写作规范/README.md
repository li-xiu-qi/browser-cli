## 核心要点速查

### 硬限制
- 标题：**20字以内**
- 单图：50-150字
- 总字数：600-1000字

### 禁用
- markdown语法（# ## - 1. 等）
- emoji
- 列表格式

### 必须用
- 段落式表达
- 第一张图交代"我怎么发现的"
- 最后一张图抛问题引互动

## 快速参考

**标题精简示例：**
```
原句：飞书CLI开源了，200+命令覆盖11个场景，OpenClaw用户有福了！
精简：飞书CLI开源，200+命令11场景，claw有福（20字）
```

**段落式表达示例：**
```
不稳：安装很简单：1. npm install 2. 配置凭证 3. 登录授权

更稳：安装很简单，npm install一行命令。然后三步搞定：配置凭证、登录授权、开始使用。
```
