# 已发布文章的临时文件

归档已发布公众号文章的 `捕获.md`、`提取.md` 等临时工作文件。

## 存放方式

每篇文章一个文件夹，以文章标题命名：

```
_archive/
├── 已发布文章临时文件/
│   ├── 别再吹"视觉压缩"了，DeepSeek-OCR的高压缩率可能误导了你/
│   │   ├── 捕获.md
│   │   ├── 提取.md
│   │   └── assets/ (可选，仅保留关键截图)
│   └── ...
```

## 归档时机

- 文章已发布到公众号
- 确认不再需要修改重发
- 临时文件在 `03_工作区` 已无活跃参考价值

## 保留原则

- 仅保留**关键素材**（最终封面图、核心图表）
- 临时截图、多轮迭代的中间图可删除
- 原始 `捕获.md`、`提取.md` 保留一份即可

## 清理周期

- 归档后满 6 个月，直接删除
- 每季度检查一次，确认无保留价值即删除
