# 归档区

本目录用于存放 xiaoke-writing skill 的临时、过期、不再使用的素材和文档。

## 使用原则

- **定期清理**：每季度检查一次，删除不再需要的文件
- **临时素材**：文章写作过程中产生的临时截图、草稿片段
- **过时版本**：已废弃的方法论版本、旧版规范
- **实验文件**：测试性质的提示词、实验记录
