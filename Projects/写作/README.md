# 写作项目

**我的个人写作工作流**，从选题到发布的完整管理。

---

## 这是什么

一个基于 **AI 协作 + 个人风格 Skill** 的写作管理系统：

- **捕获-提取-输出**：标准化的写作流程
- **风格 Skill**：持续迭代的个人写作 DNA
- **复盘机制**：每篇文章都转化为风格进化燃料

配套 AI Skill：[[.agents/skills/ai-writing-collaborator/|ai-writing-collaborator]]

---

## 快速开始

### 新文章

```bash
# 1. 在工作区创建目录
mkdir "Projects/写作/03_工作区/我的文章标题"

# 2. 按 捕获→提取→输出 流程写作
# 3. 发布后更新 04_Skill素材库/_INDEX.md
# 4. 复盘并更新风格 Skill
```

### 与 AI 协作

1. 让 AI 阅读 [[AGENTS.md]]（项目级规范）
2. 加载你的风格 Skill
3. 按流程协作写作

## 目录结构

```
Projects/写作/
├── AGENTS.md              # AI 协作规范（给 AI 看的）⭐
├── README.md              # 项目说明（本文件）
├── 01_选题思考区/          # 灵感收集（随意）
├── 02_选题库/              # 精选选题
├── 03_工作区/              # 写作工作区（核心）
│   └── [文章标题]/
│       ├── 捕获.md         # 输入：随意堆积素材
│       ├── 提取.md         # 加工：梳理大纲
│       ├── 输出.md         # 产出：最终文章
│       ├── 参考文献/
│       └── assets/
├── 04_Skill素材库/         # 风格管理与复盘
│   ├── _INDEX.md           # 文章状态索引
│   └── 复盘记录/           # 复盘文档
├── 05_参考范文/            # 值得学习的他人文章（结构/技巧）
├── 06_范文分析/            # 对参考范文的深度分析
└── 08_素材收藏/            # 观点/案例/金句收藏（内容好，形式一般）
    ├── 观点收藏/
    ├── 案例收藏/
    ├── 金句收藏/
    └── _INDEX/             # 素材索引
```

**关键原则**：
- 文章始终留在 `03_工作区/`，不移动
- 状态变化用元数据标记在 `04_Skill素材库/_INDEX.md`
- 原文唯一，通过链接引用

## 工作流程

```
灵感 → 01_选题思考区/ → 02_选题库/
                              ↓
                    03_工作区/[文章]/
                        捕获 → 提取 → 输出
                              ↓
                         发布
                              ↓
                    04_Skill素材库/
                        复盘 → 更新 Skill
```

### 1. 选题
- 在 `01_选题思考区/` 记录任何想法
- 有价值的进入 `02_选题库/`

### 2. 写作
- 在 `03_工作区/` 创建文章目录
- 使用 AI 协作，加载风格 Skill
- 按 **捕获→提取→输出** 流程

### 3. 发布
- 发布到目标平台
- 在 `04_Skill素材库/_INDEX.md` 标记"已发布"

### 4. 复盘（关键）
- 发布后 1-7 天内复盘
- 分析亮点和问题
- 更新风格 Skill

## 参考范文与素材收藏

### 内容分类标准

| 类型 | 位置 | 特点 | 处理方式 |
|------|------|------|----------|
| **参考范文** | `05_参考范文/` | 写作技巧优秀，值得学习模仿 | 全文保存 + 深度分析 |
| **素材收藏** | `08_素材收藏/` | 观点/案例/金句有价值，但形式一般 | 摘要记录 + 原文链接 |

**判断流程**:
```
抓取文章 → 判断价值 → 形式好？→ 是 → 05_参考范文（深度分析）
                   ↓
                 否 → 内容有价值？→ 是 → 08_素材收藏（简要记录）
                                 ↓
                               否 → 丢弃
```
