## 分析维度

每篇分析包含以下维度：

| 维度 | 说明 |
|------|------|
| **标题分析** | 拆解、技巧、可复用公式 |
| **结构分析** | 整体架构、开篇技巧、节奏控制 |
| **内容分析** | 核心观点、论证质量、信息密度 |
| **表达分析** | 语言风格、金句使用、情感控制 |
| **受众分析** | 目标人群、门槛评估、排斥点 |
| **可借鉴点** | 结构层、表达层、可复用模板 |
| **需避免点** | 问题清单、改进建议 |

## 新增分析流程

1. **阅读原文**: 阅读 `05_参考范文` 中的文章
2. **复制模板**: 复制 [[范文分析模板]]
3. **结构化分析**: 按模板各维度填写分析
4. **命名文件**: `YYYY-MM-DD_范文分析_[文章标题].md`
5. **更新索引**: 在本文件添加条目
