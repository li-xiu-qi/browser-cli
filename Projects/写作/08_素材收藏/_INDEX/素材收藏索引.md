# 素材收藏索引

**定位**: 收集观点、案例、金句等有价值的内容，但**写作形式不值得学习**  
**与 05_参考范文 的区别**:
- `05_参考范文`: 写作技巧优秀，值得**模仿学习**
- `08_素材收藏`: 内容有价值，但只值得**提取观点/案例**

---

## 收藏分类

| 分类 | 存放内容 | 使用场景 |
|------|----------|----------|
| **观点收藏** | 独特的观点、洞察、角度 | 写观点类文章时引用或反驳 |
| **案例收藏** | 典型的事件、故事、数据 | 作为文章论据或例证 |
| **金句收藏** | 精辟的句子、比喻、类比 | 引用或改编使用 |

---

## 管理原则

### 与范文的不同管理方式

| 维度 | 05_参考范文 | 08_素材收藏 |
|------|-------------|-------------|
| **分析深度** | 详细（结构、技巧、优缺点） | 简要（观点摘要、亮点标注） |
| **存储形式** | 全文+分析文档 | 摘要+原文链接 |
| **使用方式** | 学习写作技巧 | 提取内容素材 |
| **更新频率** | 长期维护 | 定期清理归档 |

### 入库标准

**必须满足以下至少一条**:
- [x] 观点独特，有启发性
- [x] 案例典型，可作论据
- [x] 金句精辟，值得引用

**且满足**:
- [ ] 写作形式一般（不需要学习其结构/表达）

### 禁止入库

- [ ] 观点平庸，无新意
- [ ] 内容过时，已失效
- [ ] 纯粹的情绪宣泄

## 使用流程

### 发现素材时

1. **判断价值**：观点/案例/金句是否有价值？
2. **判断形式**：写作形式是否值得学习？
   - **形式好** → 归入 `05_参考范文`，写详细分析
   - **形式一般** → 归入 `08_素材收藏`，简要记录
3. **选择模板**：观点/案例/金句
4. **快速记录**：10分钟内完成

### 写作时

1. **搜索素材**：在对应分类中查找
2. **验证时效**：确认内容仍有效
3. **改编使用**：不要直接复制，要融入自己的文章
4. **标注来源**：引用时注明出处

---

## 定期清理

每季度清理一次：

- [ ] 观点是否仍适用？
- [ ] 案例是否仍新鲜？
- [ ] 金句是否仍精辟？

**归档标准**：
- 过时内容 → 删除
- 已多次使用的内容 → 删除（已内化）
- 仍有用但用不上 → 保留
