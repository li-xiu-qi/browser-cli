# 参考范文索引

**定位**: 收集优秀文章作为写作学习与参考素材  
**原则**: 这些文章是「别人写的」，用于学习结构、技巧、表达方式，**不是**个人风格训练素材  

**配套资源**:
- `05_参考范文`: 存放**原文**（本文档）
- `06_范文分析`: 存放**分析文档** → [[Projects/写作/06_范文分析/_INDEX\|06_范文分析索引]]

## 使用方式

### 1. 学习特定技巧

```markdown
参考 05_参考范文/[分类]/[文章].md 的[具体技巧]，
帮我规划一篇关于 [主题] 的文章大纲。

我希望借鉴：
- [具体借鉴点]
```

### 2. 使用分析模板

使用 [[Projects/写作/06_范文分析/范文分析模板\|范文分析模板]] 对新文章进行结构化分析。

---

## 新增范文流程

1. **抓取文章**: 使用 Cliper 工具抓取原文
2. **分类存放**: 根据内容类型放入对应子目录
3. **填写索引**: 在本文件添加条目，标注亮点和可借鉴点
4. **分析文档**: 使用模板进行分析，存入 `06_范文分析/`
5. **更新索引**: 在 `06_范文分析/_INDEX.md` 添加分析条目
