# 写作项目

这里是您的内容生产中心。采用 **"选题库控制 + 工作区生产"** 的双层架构。

##  目录结构

### 1. 选题库 (控制层)
- **`选题中心.base`**: 全局指挥部。在这里查看所有文章的状态，通过索引链接快速跳转。
- **`待选/`**: 灵感池。存放尚未启动的碎片想法。

### 2. 工作区 (生产层)
每一篇文章都是一个独立的项目文件夹，包含标准的“三段式”文件：

- **`[文章标题]/`**
    - **`捕获.md`**: **素材箱**。存放剪藏内容、参考资料链接、原始灵感。
    - **`提取.md`**: **加工台**。存放逻辑大纲、思维导图、核心观点提炼。
    - **`输出.md`**: **成品区**。最终的成稿内容。
    - **`assets/`**: **资源库**。存放该文章专用的图片、PDF 等附件。

##  工作流

1.  **灵感**: 在 `选题库/待选` 中新建笔记记录想法。
2.  **启动**: 
    - 在 `工作区` 新建文件夹，以文章标题命名。
    - 将灵感文件移入，重命名为 `捕获.md`。
    - 创建空的 `提取.md` 和 `输出.md`。
3.  **写作**: 在 `选题中心.base` 中点击链接，分别进行素材整理、大纲梳理和正文写作。
4.  **完成**: 在 `输出.md` 的 Frontmatter 中标记 `status: 已完成`。
