## 关键信息摘要

### 发布时间线
- **2025年10月27日**: Beta 版本发布，仅限 Max、Enterprise 和 Teams 用户（1000 人等待名单）
- **2026年1月24日**: 向所有 Pro 订阅者开放

### 订阅要求
- Pro (~$20/月)
- Max
- Team
- Enterprise

### 核心功能
- 理解整个 Excel 工作簿，包括嵌套公式和多标签页依赖关系
- 附带单元格级别引用的解答
- 在保留公式的前提下更新假设条件
- 快捷键: Mac (`Control+Option+C`) / Windows (`Control+Alt+C`)

### 金融分析特性
- 连接 Moody's、LSEG (Refinitiv)、Aiera 等数据源
- 六种新的 Agent Skills 用于尽职调查和公司分析
- 在金融任务上的准确率达到 55.3%（Sonnet 4.5）
