# SVG 配图方法论框架

## 核心洞察

从 PNG 复刻项目中提炼的**三层模型**，可迁移到任何 SVG 配图场景：

```
┌─────────────────────────────────────────┐
│           SVG 配图三层模型               │
├─────────────────────────────────────────┤
│  第一层：框架层（Layout）                │
│  • 确定画布尺寸和网格系统                 │
│  • 划分功能区域                          │
│  • 用虚线框占位验证                       │
├─────────────────────────────────────────┤
│  第二层：组件层（Components）            │
│  • 识别可复用元素                         │
│  • 在 <defs> 中定义                       │
│  • 独立文件调试                          │
├─────────────────────────────────────────┤
│  第三层：样式层（Style）                 │
│  • 统一配色方案                          │
│  • 字体和排版规范                         │
│  • 视觉层次（大小、颜色、对比度）          │
└─────────────────────────────────────────┘
```

## 场景迁移矩阵

| 场景 | 适用性 | 关键组件 | 布局策略 | 配色建议 | 推荐工具 |
|------|--------|----------|----------|----------|----------|
| **技术架构图** |  适合 | 服务器图标、数据库、箭头连接线 | 左右/上下分层 | 深色背景+亮色强调 | SVG |
| **PPT 配图** |  适合 | 中心图标、关键数字、简短标签 | 中心放射/左右对比 | 跟随品牌色 | SVG |
| **文章配图** |  适合 | 拟人化角色、流程箭头、标签 | 故事性流动 | 柔和色调 | SVG |
| **数据图表** | ⚠️ 一般 | 坐标轴、柱状/饼图、图例 | 标准图表布局 | 数据色板 | **ECharts/D3** |
| **流程图** |  不适合 | 步骤框、箭头、判断菱形 | 线性/泳道布局 | 步骤渐变色 | **Graphviz** |
| **时序图** |  不适合 | 生命线、消息箭头、参与者 | 垂直布局 | 区分角色 | **PlantUML** |

### SVG 适用边界

** 适合用 SVG**：
- 静态概念图（架构图、对比图）
- 简单图标和装饰元素
- 需要精确控制样式的配图
- PPT/文章中的装饰性图表

** 不适合用 SVG**（有更好工具）：
- 复杂流程图 → **Graphviz**
- 复杂数据可视化 → **ECharts/D3.js**
- 时序图/类图 → **PlantUML/Mermaid**
- 需要交互的图表 → **D3.js + React/Vue**

## 可复用的 SVG 配方

### 配方1：左右对比型（适合 PPT）
```svg
画布: 1200x600
├─ 左侧区域: x=50, width=550
│   ├─ 标题 (y=80)
│   ├─ 核心图标 (y=250, size=200)
│   └─ 说明文字 (y=450)
├─ 右侧区域: x=600, width=550
│   ├─ 标题 (y=80)
│   ├─ 核心图标 (y=250, size=200)
│   └─ 说明文字 (y=450)
└─ VS 标识: x=600, y=300
```

### 配方2：中心放射型（适合概念图）
```svg
画布: 800x800
├─ 中心概念: x=400, y=400, r=150
├─ 分支1: 角度0°, 距离250px
├─ 分支2: 角度72°, 距离250px
├─ 分支3: 角度144°, 距离250px
├─ 分支4: 角度216°, 距离250px
└─ 分支5: 角度288°, 距离250px
```

### 配方3：流程时间线（适合步骤说明）
```svg
画布: 1000x400
├─ 时间线: y=200, x=100 to x=900
├─ 步骤1: x=150, 圆形+标签
├─ 步骤2: x=350, 圆形+标签
├─ 步骤3: x=550, 圆形+标签
└─ 步骤4: x=750, 圆形+标签
```

## 设计系统 Token

### 间距系统
```
4px   - 最小间隙（图标与文字）
8px   - 紧凑间距
16px  - 标准间距
24px  - 宽松间距
40px  - 区域分隔
```

### 字体系统
```
标题: 24-32px, bold
副标题: 18-20px, medium
正文: 14-16px, normal
标签: 12px, normal
```

### 阴影层级
```
Level 1: 卡片、按钮 → dx=0, dy=2, blur=4, opacity=0.1
Level 2: 弹窗、对话框 → dx=0, dy=4, blur=8, opacity=0.15
Level 3: 浮层、提示 → dx=0, dy=8, blur=16, opacity=0.2
```

## 场景化组件库

### 通用图标（拟物化）
- 电脑/服务器
- 数据库（圆柱体）
- 云朵（云服务）
- 手机/平板
- 人物头像

- 箭头（直线、曲线、双向）
- 连接线（实线、虚线、带箭头）

### 数据可视化
- 进度条
- 饼图/环形图
- 柱状图
- 折线图（简化版）

- [ ] 划分功能区域

- [ ] 识别可复用组件
- [ ] 在 `<defs>` 中定义
- [ ] 独立文件调试验证

- [ ] 更新组件库
- [ ] 补充到方法论文档
