# 字体资源说明

本目录存放 Google Noto 字体系列的 CJK 简体中文版本，用于文档渲染和导出。

## 字体列表

| 文件名 | 中文名 | 字重 | 大小 | 用途 |
|--------|--------|------|------|------|
| NotoSansCJKsc-Bold.otf | 思源黑体 | 粗体 | 16.2 MB | 标题、强调文字 |
| NotoSansCJKsc-Light.otf | 思源黑体 | 细体 | 15.6 MB | 注释、辅助文字 |
| NotoSansCJKsc-Medium.otf | 思源黑体 | 中等体 | 15.8 MB | 正文、常规内容 |
| NotoSerifCJKsc-Regular.otf | 思源宋体 | 常规体 | 23.4 MB | 书籍、正式文档 |

## 字体简介

**思源黑体 Noto Sans CJK SC**

- 由 Google 与 Adobe 合作开发
- 支持简体中文、繁体中文、日文、韩文
- 无衬线设计，适合屏幕显示
- 提供三种字重：Light、Medium、Bold

**思源宋体 Noto Serif CJK SC**

- 与思源黑体配套的有衬线字体
- 适合长文阅读与正式文档
- 具有传统印刷字体的典雅风格

## 使用场景

- Markdown 转 PDF 导出
- 图表生成（Mermaid、PlantUML 等）
- 文档排版与打印
