## 处理流程

```
原始电子书 (PDF/EPUB)
    ↓
[转换工具] → Markdown
    ↓
00_Inbox-待分类/           # 临时存放
    ↓
质量评估 → 分类整理
    ↓
AI协作学/心理学/经济学/...  # 最终位置
```
