# 常用软件工具清单

## 阅读与信息处理
- **简悦 (SimpRead)**
  - **简介**: 网页沉浸式阅读工具。
  - **核心**: 消除干扰、稍后读、支持导出 Markdown/PDF。
  - **地址**: [官网地址](https://www.yuque.com/kenshin/simpread/uc4g3g)

## 效率与安全
- **KeePass**
  - **简介**: 开源、本地优先的密码管理工具。
  - **核心**: 离线存储、数据库加密、自主可控、插件生态丰富。
  - **地址**: [下载页面](https://keepass.info/download.html)

## 素材与数字资产管理
- **Eagle**
  - **简介**: 设计师必备的素材管理工具。
  - **核心**: 跨平台采集、标签/颜色搜索、本地存储、支持音视频字体。
  - **地址**: [下载页面](https://cn.eagle.cool/download)

## 内容下载与数据采集
- **DataTool.vip**
  - **简介**: 社媒内容下载与轻量数据采集工具站。
  - **核心**: 支持粘贴链接下载 YouTube、TikTok、小红书、抖音、B站等平台内容，也提供浏览器扩展做批量下载和 Google Maps 商家信息提取。
  - **提醒**: 完整能力依赖浏览器扩展，使用时要自己判断版权边界和站点规则。
  - **地址**:
    - [官网地址](https://www.datatool.vip/)
    - [插件安装说明](https://www.datatool.vip/downloader/install-plugin/zh)

## 知识管理与笔记
- **Obsidian**
  - **简介**: 基于 Markdown 的本地知识管理软件。
  - **核心**: 双向链接、可视化图谱、本地优先、极强插件生态（第二大脑）。
  - **地址**: [官网地址](https://obsidian.md/)

## AI 编程与 Agent
- **Qwen Code**
  - **简介**: 专为 Qwen3-Coder 优化的终端 AI Agent (CLI)，支持 Agentic Workflow。
  - **核心**: 终端交互、Skills/SubAgents 扩展、Plan Mode 规划、OpenAI API 兼容。
  - **安装**: `npm install -g @qwen-code/qwen-code@latest`
  - **地址**: 
    - [GitHub 仓库](https://github.com/QwenLM/qwen-code)
    - [官方文档 (中文)](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)

- **Pencil**
  - **简介**: 集成在 IDE 中的画布设计工具，支持 AI 生成 UI 和代码。
  - **核心**: 设计与代码一体化、Git 版本控制、AI 驱动的 Canvas、支持导入 Figma。
  - **地址**: 
    - [官网地址](https://www.pencil.dev/)
    - [Prompt 库](https://www.pencil.dev/prompts)
