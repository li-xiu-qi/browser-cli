# AI 数据分析工具索引

整理 AI 驱动的数据库交互、Text-to-SQL 及数据分析工具清单。

## AI 数据库交互与 Text-to-SQL

### DB-GPT

- **简介**: 全能型框架，集成多模型管理、RAG、Text-to-SQL 及 Agent 工作流编排
- **源码仓库**: [https://github.com/eosphoros-ai/DB-GPT](https://github.com/eosphoros-ai/DB-GPT)
- **Star**: 18.1k

### Vanna AI

- **简介**: 基于 RAG 模式索引元数据的 Python 框架，专注高准确率 SQL 生成与集成
- **源码仓库**: [https://github.com/vanna-ai/vanna](https://github.com/vanna-ai/vanna)
- **Star**: 22.5k

### SQLCoder

- **简介**: Defog 开源的 SOTA 级 Text-to-SQL 专用大语言模型系列
- **源码仓库**: [https://github.com/defog-ai/sqlcoder](https://github.com/defog-ai/sqlcoder)
- **Star**: 4.0k

### SuperSonic

- **简介**: 腾讯音乐开源的 Chat BI 平台，融合 Headless BI 与自然语言查询
- **源码仓库**: [https://github.com/tencentmusic/supersonic](https://github.com/tencentmusic/supersonic)
- **Star**: 4.7k
- **文档**: [https://supersonicbi.github.io/](https://supersonicbi.github.io/)

### SQLBot

- **简介**: DataEase 出品的基于大模型和 RAG 的智能问数系统
- **源码仓库**: [https://github.com/dataease/SQLBot](https://github.com/dataease/SQLBot)
- **Star**: 5.4k

### WrenAI

- **简介**: 主打语义建模层的 AI 引擎，通过预定义业务逻辑解决复杂多表关联查询
- **源码仓库**: [https://github.com/Canner/WrenAI](https://github.com/Canner/WrenAI)
- **Star**: 13.8k

### Chat2DB

- **简介**: 带 AI 辅助功能的数据库管理客户端，可直接作为 Navicat 的 AI 增强替代品
- **源码仓库**: [https://github.com/chat2db/Chat2DB](https://github.com/chat2db/Chat2DB)
- **Star**: 25.1k

### SQLChat

- **简介**: 纯对话式的轻量级 SQL 交互工具，提供类似 ChatGPT 的数据库沟通体验
- **源码仓库**: [https://github.com/sqlchat/sqlchat](https://github.com/sqlchat/sqlchat)
- **Star**: 5.7k

### DuckDB-NSQL-7B

- **简介**: MotherDuck 与 Numbers Station 联合推出的 DuckDB 专用 Text-to-SQL 模型
- **博客介绍**: [https://motherduck.com/blog/duckdb-text2sql-llm/](https://motherduck.com/blog/duckdb-text2sql-llm/)

### chatDuck

- **简介**: 用自然语言与 DuckDB 数据库对话的交互式工具
- **源码仓库**: [https://github.com/kyaukyuai/chatDuck](https://github.com/kyaukyuai/chatDuck)
- **Star**: 11

### PremSQL

- **简介**: 端到端本地优先的 Text-to-SQL 库，支持使用小语言模型构建安全的数据库查询
- **源码仓库**: [https://github.com/premAI-io/premsql](https://github.com/premAI-io/premsql)
- **Star**: 431

## AI 原生数据分析与 Agent

### Data Formulator

- **简介**: 微软研究院出品的 AI 驱动数据可视化工具，通过概念驱动的方式创建丰富图表
- **源码仓库**: [https://github.com/microsoft/data-formulator](https://github.com/microsoft/data-formulator)
- **Star**: 14.8k

### Evidence

- **简介**: 代码优先的 BI 框架，用 SQL 和 Markdown 构建数据产品和仪表盘
- **源码仓库**: [https://github.com/evidence-dev/evidence](https://github.com/evidence-dev/evidence)
- **Star**: 5.9k
- **官网**: [https://evidence.dev/](https://evidence.dev/)

### PandasAI

- **简介**: 为 Pandas 赋予对话能力，直接用自然语言对 DataFrame 进行数据清理和提问
- **源码仓库**: [https://github.com/gventuri/pandas-ai](https://github.com/gventuri/pandas-ai)
- **Star**: 23.1k

### Open Interpreter

- **简介**: 开源版代码解释器，允许 AI 在本地环境执行 Python 代码完成复杂数据任务
- **源码仓库**: [https://github.com/OpenInterpreter/open-interpreter](https://github.com/OpenInterpreter/open-interpreter)
- **Star**: 62.0k

### PyGWalker

- **简介**: 将数据分析转化为类似 Tableau 的交互式界面，支持在 Jupyter 中通过 AI 指令探索数据
- **源码仓库**: [https://github.com/Kanaries/pygwalker](https://github.com/Kanaries/pygwalker)
- **Star**: 15.6k

### LIDA

- **简介**: 微软出品的 AI 图表生成引擎，能自动为数据推荐分析方案并生成多类型可视化
- **源码仓库**: [https://github.com/microsoft/lida](https://github.com/microsoft/lida)
- **Star**: 3.2k

### TaskWeaver

- **简介**: 微软开发的面向数据任务的智能体框架，支持自主规划并调用 Python 库处理复杂数据
- **源码仓库**: [https://github.com/microsoft/TaskWeaver](https://github.com/microsoft/TaskWeaver)
- **Star**: 6.1k

### RATH

- **简介**: 自动化的增强分析工具，专注于数据模式的自动发现、因果分析及可视化仪表盘生成
- **源码仓库**: [https://github.com/Kanaries/RATH](https://github.com/Kanaries/RATH)
- **Star**: 4.6k

### AI Data Science Team

- **简介**: 专门用于数据科学工作流的 Python Agent 库，包含 AI Pipeline Studio 可视化工具
- **源码仓库**: [https://github.com/business-science/ai-data-science-team](https://github.com/business-science/ai-data-science-team)
- **Star**: 4.7k

### DataMind

- **简介**: 用于自动化科学发现的数据分析 Agent，支持多步推理和复杂数据分析任务
- **源码仓库**: [https://github.com/zjunlp/DataMind](https://github.com/zjunlp/DataMind)
- **Star**: 63

### DeepAnalyze

- **简介**: 首个面向自治数据分析的 Agentic LLM，模型、代码、训练数据全开源
- **源码仓库**: [https://github.com/ruc-datalab/DeepAnalyze](https://github.com/ruc-datalab/DeepAnalyze)
- **Star**: 3.6k

### LAMBDA

- **简介**: 无代码多智能体数据分析系统，利用大模型能力实现自动化的数据洞察
- **源码仓库**: [https://github.com/AMA-CMFAI/LAMBDA](https://github.com/AMA-CMFAI/LAMBDA)
- **Star**: 551
