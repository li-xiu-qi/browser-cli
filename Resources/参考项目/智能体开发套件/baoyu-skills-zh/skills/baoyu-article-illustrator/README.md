# baoyu-article-illustrator (文章配图专家)

这是一个专门为文章自动分析并生成高直觉、高一致性配图的 AI 智能体。它不仅能识别文章中需要视觉辅助的位置，还能通过独特的“类型 × 风格”二维模型，确保生成的图片既能增强信息传达，又能在视觉上保持风格统一。

## 🌟 核心特性

- **高直觉配图逻辑**：拒绝字面意义的堆砌（如提到“电锯切西瓜”不会真的画西瓜），而是深层理解文章的**核心论点**和**抽象概念**进行可视化。
- **二维控制模型 (Type × Style)**：
    - **类型 (Type)**：控制信息结构（如信息图、场景、流程图、对比图、架构图等）。
    - **风格 (Style)**：控制视觉美感（如 Notion 风、优雅、温暖、极简、蓝图、水彩等）。
- **自动化工作流**：从内容分析、位置识别、参数确认到提示词生成及文章自动更新，提供全链路支持。
- **高度可定制**：通过 `EXTEND.md` 支持自定义水印、偏好风格、语言及输出路径。

## 🛠️ 技术原理

### “类型 × 风格”矩阵
智能体根据文章的“内容信号”自动匹配最合适的组合：
- **技术/指标类**：`infographic` + `blueprint/notion`
- **叙事/情感类**：`scene` + `warm/watercolor`
- **教程/步骤类**：`flowchart` + `notion/minimal`
- **架构/模型类**：`framework` + `blueprint/notion`

### FDW (Foreign Data Wrappers) 式集成
该工具设计上支持灵活的输出管理，可以选择：
1. **同级目录**：直接在文章所在目录生成图片。
2. **子目录 (imgs/illustrations)**：保持项目整洁。
3. **独立目录**：按主题 slug 统一管理所有配图。

## 📂 项目结构

```text
baoyu-article-illustrator/
├── SKILL.md                # 技能核心定义（流程、规则、矩阵）
├── prompts/
│   └── system.md           # 核心绘图提示词规范（16:9, 手绘感, 文字风格）
├── references/             # 参考资源（风格库、配置规范等）
└── illustrations/          # 默认生成的配图输出目录（示例）
```

## 🚀 快速开始

1. **分析文章**：将文章路径或内容提供给智能体。
2. **确认配置**：根据智能体建议确认配图密度（精简/均衡/丰富）和视觉风格。
3. **执行生成**：智能体会自动生成 `outline.md` 规划和 `prompts/` 提示词。
4. **自动更新**：智能体可直接修改原 Markdown 文件，在对应位置插入 `![description](path)`。

## 📝 提示词设计规范

本项目生成的图片均遵循以下原则：
- **手绘感 (Hand-drawn)**：拒绝摄影写实，提升亲和力。
- **比例**：统一 16:9 横屏。
- **文字一致性**：图片内文字与文章语言一致，且字体风格与插画融合。
