Reference documentation for the [`langchain-core`](https://pypi.org/project/langchain-core/) package.

`langchain-core` contains the core interfaces and abstractions used across the LangChain ecosystem. Most users will primarily interact with the main [`langchain`](Projects/内容创作/草稿箱/docs/参考文献/docs/reference/python/docs/langchain/langchain/index.md) package, which builds on top of `langchain-core` and provides implementations for all the core interfaces.
