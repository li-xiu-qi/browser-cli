# Obsidian 常用插件清单

## 1. 基础与界面增强
- **Default New Tab Page**: 设置新建标签页时默认打开的页面。
- **Homepage**: 设置每次启动 Obsidian 时默认打开的首页。
- **Style Settings**: 提供图形化界面，用于深入自定义支持该插件的主题样式。
- **Quiet Outline**: 强力的大纲插件，支持搜索、筛选，以及通过拖拽标题直接修改笔记结构（双向同步）。
- **Floating TOC**: 在笔记侧边显示浮动目录，滚动时长篇笔记时会自动高亮当前章节。
- **Plugin Groups**: 插件分组管理，可根据场景（如写作模式、绘图模式）一键开启/关闭特定插件组。

## 2. 编辑与输入效率
- **Various Complements**: 智能自动补全，根据库中已有词汇弹出建议，类似代码编辑器的输入体验。
- **Easy Typing**: 格式化输入，支持自动中英文间距、关闭英文首字母大写等功能。
- **Auto Link Title**: 粘贴链接时自动抓取网页标题并转化为 Markdown 格式。
- **Linter**: 自动化格式检查与修正，可一键执行如“标题后加空行”、“删除多余空行”等规则，保持笔记统一。
- **Editing Toolbar**: 顶部可视化工具栏，提供类似 Word 的加粗、高亮、对齐等快捷操作。

## 3. 文件与附件管理
- **Attachment Management**: 附件自动化管家，根据笔记名称和路径自动重定向并整理附件到指定文件夹。
- **Image Converter**: 强大的图片处理工具，支持在库内直接转换（WEBP/JPG/PNG）、压缩、裁剪、翻转图片。
- **Local Image Plus**: 自动将笔记中的网络图片下载并保存到本地，防止素材失效。
- **Clear Unused Images**: 自动扫描并删除库中未被引用的冗余图片。
- **File Cleaner Redux**: 冗余文件清理工具，支持通过配置后缀名清理各类不使用的附件文件。

## 4. 知识连接与组织
- **Smart Connections**: 利用 AI Embedding 技术发现笔记间的语义关联，辅助内容链接。
- **Tags Wrangler**: 标签重构利器，支持全局重命名、合并标签以及创建标签专属页面。
- **Colored Tags**: 为标签添加颜色视觉区分，一眼识别不同主题或状态的笔记。
- **Auto Note Mover**: 自动化归档，根据笔记内的标签自动将其移动到预设的文件夹中。

## 5. 导出与发布
- **Pandoc Plugin**: 导出全能王，将 Markdown 转换为 Word (DOCX)、电子书 (ePub) 或 PDF 等多种格式。
- **Export Image**: 将 Markdown 笔记直接渲染成精美的长图，便于社交媒体分享。
- **Wechat Public Platform**: 将文章或视频资源直接发布至微信公众号后台。
- **MP Preview / NoteToMP**: 优化微信公众号的发布流程，支持格式美化与快速上传。

## 6. 特定场景插件
- **学术型**
  - **Pandoc Reference List**: 学术写作助手，在侧边栏即时显示 @ 引用标记对应的格式化参考文献。
- **日记型**
  - **Thino**: 灵感卡片式日记管理工具，支持快速记录与日历视图回顾。
- **绘图型**
  - **Excalidraw**: 顶级的白板绘图插件，支持手写感绘图、脚本扩展及 CSS 美化，适合梳理思路。
