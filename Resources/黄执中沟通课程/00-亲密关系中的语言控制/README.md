### 📚 课程核心框架

黄执中提出亲密操控通过**四种力量**实现：

1. **引发罪恶感** — 让你觉得自己欠对方的
2. **打击价值感** — 让你觉得自己没资格反对
3. **破坏现实感** — 让你怀疑自己的记忆和感知
4. **削弱安全感** — 让你只能依赖操控者

**核心洞察**：操控需要双方配合，真正的权力始终在被操控者手中。受害者不离开是因为有"贪恋"：
- 贪恋被强者保护
- 贪恋逃避责任
- 贪恋用服从来换取肯定
- 贪恋"受害者红利"
