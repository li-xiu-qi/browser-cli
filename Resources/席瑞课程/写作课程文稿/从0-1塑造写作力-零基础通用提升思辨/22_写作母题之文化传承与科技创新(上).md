# 22_写作母题之文化传承与科技创新（上）

## 1. 课程导入：软实力与硬实力的交织
文化是软实力，科技是硬实力。作为新时代的写作者，我们需要关注文化对外输出与文化自信，以及科技硬实力的提升，从而构建对国家整体发展的深度理解。

> **核心观点**：你的生命与敦煌、阳关、三峡、天山有关，与唐诗宋词有关，这就是文化自信的根源。

---

## 2. 文化类话题写作：寻找“中国味”
文化类作文往往显得宏大而抽象，核心在于如何通过具体的载体将其“写活”。

### 2.1 案例分析：《中国味》
- **结构**：回眸历史 -> 仰望苍穹 -> 总结人格。
- **意象化回应**：
  - **无尽含蓄的等待**：李清照（帘卷西风，人比黄花瘦）。
  - **安之若素的淡泊**：苏轼（一蓑烟雨任平生，竹杖芒鞋）。
  - **不惧生死的豪迈**：项羽（死亦为鬼雄）。
  - **忠肝义胆的信念**：文天祥（我心一片磁石针）。
- **点评**：文章将抽象的“中国味”落位到了具体的诗词与历史人物的命运中，通过“诗词寓人格”的方式，展现了民族文化的厚重积淀。

### 2.2 文化类作文的三大实操建议
1.  **化虚为实，具象承载**：文化无所不包，建议落位到你最熟悉的**诗词**上。诗词是教材中现成的矿山，最易上手。
2.  **以小见大，连接现实**：拒绝“空对空”。要将古人的智慧与当下的社会现象（如浮躁风气、逐利心理）相连接，发出时代的声音。
3.  **典雅精致，突出文化感**：适当加入文言词、四字成语及名家散文的化用（如余秋雨、汪曾祺）。

## 3. 素材库扩充：家风与民俗

### 3.1 家风题材：连接个人与国家的纽带
推荐素材：系列剧《家道颖颖》。
- **核心价值观**：忠孝、仁爱、信义、和平。
- **典型案例**：张家几代人（抗美援朝的张承仁、排雷牺牲的周祥龙、战疫一线的周杨）。他们用百年的家风传承，诠释了“家是最小国，国是千万家”。

### 3.2 传统节日与民间文化
- **传统节日**：关注清明（寒食）、七夕（乞巧）、元宵的由来与习俗。
- **民间文化（非遗）**：
  - 鲁迅《社戏》。
  - 汪曾祺《端午的鸭蛋》（民间风味与生活趣味）。
  - 沈从文《云南的歌会》（民族艺术的独特性）。

## 4. 总结：做文化的守望者
1.  **多读**：读苏东坡、读魏晋风度（《世说新语》）、读名家散文。
2.  **多看**：关注传统文化习俗与人文艺术。
3.  **多行**：在旅行中记录所见所闻，用文字留下永久的文化记忆。
