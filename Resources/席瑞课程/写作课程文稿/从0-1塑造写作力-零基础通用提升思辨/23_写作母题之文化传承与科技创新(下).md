# 23_写作母题之文化传承与科技创新（下）

## 1. 课程导入：硬实力的跨越
科技是国家的硬实力。近年来，我国在多个尖端领域取得了令世界瞩目的突破。作为写作者，我们需要具备“仰望星空”的视野，将这些硬核科技转化为文章中的逻辑与力量。

### 近年重大科技成果概览
- **量子计算**：“九章”量子计算机原型机，实现量子优越性，使我国成为全球第二个实现该目标的国家。
- **探月工程**：“嫦娥五号”实现五项中国首次（地外采样封装、点火起飞、轨道对接、近第二宇宙速度返回、月壤分析系统）。
- **空间探测**：
  - “天和”核心舱：彰显“特别能吃苦、特别能战斗”的航天精神。
  - “天问一号”与“祝融号”：标志着我国首次火星探测任务圆满成功。
- **全球组网**：北斗三号系统完成部署，55 颗卫星构成全球服务的“天网”。

## 2. 科技类议论文：从“重器”到“哲思”
科技类作文不只是在写技术，更是在写规律与人文。

### 2.1 案例分析：《垒土之妙，九仞之基》
- **逻辑跳跃**：由北斗系统的“55 颗卫星各司其职” -> 联想到人事领域的“人尽其能，物尽其用”。
- **引用支撑**：引用《楚辞·卜居》“物有所不足，智有所不明”论证分工协作的重要性。
- **现实连接**：结合火神山医院建设（中国速度、凝聚力大网）展示协作的威力。
- **点评**：文章打通了科技、历史与现实，展现了极广的知识面和深刻的规律总结能力。

### 2.2 科技类作文的三大核心建议
1.  **仰望星空**：了解并列举科技发展的巨大成果（如 5G、人工智能、航天追梦人），增强文章的时代感。
2.  **脚踏实地**：对未来做出务实的展望，不要让文章显得悬浮。
3.  **关注人文与思辨**：科技最终是为人类服务的。要关注科技与文化、科技与身心健康（如短视频奴役、AI 伦理）的辩证关系。

## 3. 科技母题名言与素材积累

### 3.1 关于“旧与新”的辩证
- **刘禹锡**：“芳林新叶催陈叶，流水前波让后波。”
- **康有为**：“新则壮，旧则老；新则鲜，旧则腐；新则通，旧则滞。”

### 3.2 关于科技的定位（官方视角）
- **国之利器**：科技是国家赖之以强、企业赖之以赢、人民生活赖之以好的工具。
- **批判性思维**：如果人们的生活品质和健康未因科技改善，则是被科技奴役。

### 3.3 5G 助力抗疫素材
- **应用场景**：线上办公、云网融合、5G 医疗机器人、红外热成像测温。
- **主题连接**：体现科技在极端考验下的“守护”力量。

## 4. 总结：打通文化与科技的任督二脉
1.  **文化（软实力）**：化虚为实，落位诗词，追求典雅精致。
2.  **科技（硬实力）**：仰望星空，关注应用，追求辩证思考。

## 5. 实践练习建议
以《文明的韧性》为题写一篇议论文。
- **思考路径**：可以从中国历史的变迁、文学艺术的积淀、以及面对现代科技冲击时传统品格的坚守等角度，探讨中华文明为何能历经风雨而绵延至今。
