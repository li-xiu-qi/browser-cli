# 08_思辨与立意的核心技法（上）

## 1. 课程导入：从记叙文到议论文的跨越
议论文是比记叙文更高阶的版本。记叙文侧重**情感表达**，而议论文则侧重**逻辑思辨**。

议论文的难点在于审题。如果题目审错了、立意偏了，后续的论点和论据写得再好也是徒劳。因此，在动笔写议论文之前，必须先学会如何读、如何审、如何想。

### 案例对比：记叙文 vs 议论文
- **记叙文（青铜级）**：如史铁生的《秋天的怀念》。题目通常明确（如《那个星期天》），内容清晰，有完整的起因、经过、结果。
- **议论文（王者级）**：如《红楼梦》“大观园试才题对额”。
  - **材料**：众人题名，有人直接移用（欧阳修），有人借鉴化用（贾政），有人根据情境独创（贾宝玉）。
  - **核心**：不是考查红楼梦知识，而是让你分析“移用、化用、独创”这三种做法在广泛领域中的启示。

## 2. 审题的正误判断：抽离普遍性
审题的多元角度不代表没有对错。最常见的错误是**拘泥于材料的特殊性**，而忽略了**普遍性**。

### 练习：卖不出去的豆子
> 豆子 -> 豆芽 -> 豆苗 -> 盆景 -> 移植地里结新豆。

- **错误立意**：犹太人是世界上最聪明的民族。（过于纠结材料背景，未抽离普遍价值）
- **正确立意**：
  - 善用头脑，将危机化为转机。
  - 做人做事不能死板，要学会变通。
  - 面对困难，心里永不言弃，总能走出新路。

## 3. 立意的高下：千古文章“意”为高
立意就像统帅。优秀的立意能决定一篇文章的天花板。

### 立意分级示例：我的爷爷是农民
- **青铜（小学级）**：我的爷爷很勤劳，我爱他。（直抒胸臆，太低阶）
- **白银（初中级）**：劳动最光荣，赞美劳动人民群体。（从个人上升到群体）
- **黄金（高中/进阶级）**：土地承载着中华民族生生不息的血脉，农人血脉与土地的深层连接。（上升到民族性、文化高度，以小见大）

## 4. 实战拆解：如何建立正确的审题思维？

### 案例：“双奥之城”
- **左侧表格**：对比 08 年与 22 年在成绩、群众体育、科技、交通、经济五维度的跨越。
- **右侧文字**：点明综合国力的发展，见证从懵懂儿童向有为青年的跨越。

#### 核心思维：时间维度轴
1.  **立足当下**：2022 年冬奥会，青年人的使命与荣耀。
2.  **回溯过往**：2008 年北京奥运，5000 年文明的传承与坚持。
3.  **展望未来**：融入民族复兴，坚持创新与卓越。

> **结论**：好的审题要学会把时间维度拉长。在变迁中寻找“变”与“不变”。**变的是科技与国力，不变的是自信与创新的精神。**

## 5. 总结：议论文审题三部曲
1.  **抓住总结句**：通过复杂材料提取核心题眼。
2.  **拒绝小众偏难**：选择符合主流价值观、具有普遍意义的角度。
3.  **逐层拔高立意**：从个人到群体，从现状到历史，从现实到哲学。
