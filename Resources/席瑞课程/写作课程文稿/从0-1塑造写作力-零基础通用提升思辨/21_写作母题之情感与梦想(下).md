# 21_写作母题之情感与梦想（下）

## 1. 成长与梦想：励志类主题的进阶
成长梦想类是写作中最为昂扬、积极的部分。它不仅关乎个人的志趣，更关乎时代赋予我们的使命。

### 1.1 案例分析：从“追星”看价值观的转变
- **起因**：起初追逐衣着华丽、颜值爆表的娱乐明星。
- **转折**：疫情期间，看到逆行的钟南山院士、脸上勒出血痕的医护人员、守卫社区的党员志愿者。
- **升华**：
  > “追星就追这样的星：当国家危难时挺身而出，没有明星光环却能遮风挡雨，在黑暗中让我们看到希望。”
- **点评**：文章通过态度的反转和有力的排比，将一个普通的“追星”话题拔高到了信仰与责任的高度。

### 1.2 两个讨论维度
1.  **个人梦想**：初心、激情、对某一领域的专业热爱。
2.  **时代使命**：生逢盛世的担当，对社会、国家的感恩与反哺。

## 2. 写好情感美文的三大法宝

### 2.1 真情动人：拒绝空洞的说教
只有真正动情的人，才能看到细节。
- **负面示范**：“逛超市是一件幸福的事。”（太空洞）
- **正面示范**：描写小时候去超市前的期待、抢推车的兴奋、趁妈妈不注意往车里塞零食的小动作。
- **原则**：文章是为情而作，真情是装不出来的。

### 2.2 见微知著：小处着手，大处落笔
拒绝喊口号，寻找那个能承载宏大情感的微小载体。
- **案例**：与其说“祖父很伟大”，不如写他每次送别时站在村口那棵**老槐树**下久久张望的身影。

### 2.3 发掘与积累：生活处处皆文章
学会用“导演”的眼睛看世界。
- **素材来源**：
  - **老照片**：从中读出历史与故事（如学骑车的孩子、被遗忘的吉他）。
  - **生活细节**：如胡同里的物件、路边的标语。
  - **经典作品**：如动漫《中国奇谭》，从生活细节中挖掘剧本灵感。

---

## 3. 总结：情感与梦想的闭环
1.  **类型归纳**：写景状物（托物言志）、爱与时光（亲情怀旧）、成长梦想（励志担当）。
2.  **核心技法**：
    - 典型象征与推陈出新。
    - 真情实感与意象寄托。
    - 见微知著与时代连接。

## 4. 实践练习建议
以《散步》为题写一篇文章。
- **要求**：不限体裁（写景、叙事、成长皆可），字数 600 字以上。尝试运用本课所学的“见微知著”法，找到一个具体的意象作为全文的灵魂。
