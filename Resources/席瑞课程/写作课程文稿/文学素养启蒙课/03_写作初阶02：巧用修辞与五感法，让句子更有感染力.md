# 写作初阶02：巧用修辞与五感法，让句子更有感染力

## 一、 复习回顾：句子的基础
在上一讲中，我们学习了句子的**完整性**（[谁/物] + [做什么/是什么/怎么样]）和**通顺性**（语序正确、符合逻辑）。

本讲我们将进阶学习两个大招，让句子变得生动优美：**修辞法**与**五感法**。

## 二、 进阶招式一：修辞法（拟人与比喻）

### 1. 拟人法：赋予事物以人的生命
拟人就是把没有生命的物体写得像人一样。
*   **方法 A：赋予动作**
    *   基础句：`湿润的东风吹过。`
    *   拟人句：`湿润的东风走过荒野，在竹林中吹着口笛。`
*   **方法 B：赋予情感**
    *   拟人句：`湿润的东风走过森林，快活地弹着曲子。`

> **练习示例**：
> *   鸟儿叫 -> `小鸟在树枝上蹦跳，叽叽喳喳唱着歌。`
> *   太阳升起 -> `红彤彤的太阳羞红了脸，微笑着升起来了。`

### 2. 比喻法：建立具体的形象连接
比喻是将一个事物（本体）比作另一个更具象的事物（喻体）。
*   **公式**：`本体` + `喻词（像、好像、仿佛）` + `喻体`
*   **案例**：`孩子们像春天的花朵一样可爱。`

> **练习示例**：
> *   雪花 -> `像柳絮、像鹅毛、像梨花。`
> *   星星 -> `像漫天飞舞的萤火虫。`
> *   校园热闹 -> `像菜市场一样。`

## 三、 进阶招式二：五感法（视听嗅味处）

很多时候写流水账是因为没打开感官。我们可以调用身体的五种感官：

| 感官 | 对应能力 | 描写要点 | 案例（写西瓜） |
| :--- | :--- | :--- | :--- |
| **视觉** | 看到 | 颜色、形状、动作 | 绿油油的外皮，像胖娃娃穿了迷彩服。 |
| **听觉** | 听到 | 声音、节奏 | 拍一拍“咚咚”响，切开时“咔嚓”一声。 |
| **嗅觉** | 闻到 | 气味、清香 | 淡淡的瓜皮清香，甜甜的果香绕着鼻尖。 |
| **味觉** | 尝到 | 酸甜苦辣、滋味 | 甜津津、凉丝丝，流进了心田。 |
| **触觉** | 摸到 | 质感、温度 | 滑滑的、凉凉的，从指尖传来舒服的感觉。 |

## 四、 总结与课后作业

### 课程小结
*   **修辞法**：通过拟人和比喻，让事物更具体、可感、亲切。
*   **五感法**：通过视、听、嗅、味、处，全方位提升写作的感受力。

### 课后作业
请选择你最喜欢的一种**植物**，尝试结合**修辞法**或**五感法**写一个生动的句子。
