# 看图写话01：掌握六要素与脑补法，把图片变故事

## 一、 挑战：如何处理多幅图片的看图写话？
很多同学看到多幅图片罗列（比如 8 幅、12 幅）就会头大。其实，只要掌握了秘诀，图片再多也不怕。

## 二、 秘诀一：要素齐全（逻辑整理）
写好故事的第一步是整理信息。任何一个完整的故事都必须具备**六要素**：

| 要素 | 说明 | 案例分析（鸭妈妈救子） |
| :--- | :--- | :--- |
| **时间** | 故事发生的时间 | 某一个白天。 |
| **地点** | 故事发生的地点 | 马路旁的下水道边。 |
| **人物** | 故事中的角色 | 鸭妈妈、三个鸭宝宝、光头老爷爷。 |
| **起因** | 故事因何而起 | 鸭妈妈带队，两只鸭宝宝失足掉进下水道。 |
| **经过** | 故事的发展过程 | 鸭妈妈发现并着急，向老爷爷求助救援。 |
| **结果** | 故事的结局 | 老爷爷成功救出鸭宝宝，皆大欢喜。 |

> **提示**：通过这六个框框，繁杂的图片瞬间就会变得清晰简单。

## 三、 秘诀二：发挥想象（脑补细节）
图片是静止的，且图片之间往往省略了大量过程。我们需要通过“脑补”来使衔接更自然：

### 1. 补全动作与神态
图片只给了一个“张大嘴”的动作，我们可以想象：
*   鸭妈妈是否在焦急地转圈圈？
*   她是否在不停扑棱翅膀吸引路人注意？

### 2. 补全语言与心理
图片里没有对话，我们可以想象：
*   鸭妈妈是如何指挥下水道里的小鸭子齐声尖叫的？
*   她是如何咬住老爷爷的裤腿不放的？

### 3. 补全过程细节
*   老爷爷是用手捞还是用工具？
*   小鸭子被救起后是否感动的喜极而泣？

## 四、 总结：要素与想象的结合

1.  **要素齐全**：搭好故事的骨架。
2.  **发挥想象**：填满故事的血肉，让它更真实、可信、顺滑。

> **注意**：六要素并不需要平均用力。在上面的案例中，“地点（下水道）”和“经过（求助与救援）”是重点，需要大笔墨描写；而“时间”则可以一笔带过。
