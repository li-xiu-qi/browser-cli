## 使用场景

### 1. MCP Server 配置
用于配置 Kimi CLI 的百度网盘 MCP Server：

```bash
kimi mcp add --transport http baidu-netdisk \
  "https://mcp-pan.baidu.com/sse?access_token=YOUR_ACCESS_TOKEN"
```

### 2. 获取 Access Token

需要通过 OAuth 流程获取 access_token：

1. 访问 [百度网盘开放平台](https://pan.baidu.com/union)
2. 使用上述 AppKey 进行 OAuth 授权
3. 获取 `access_token` 后填入项目 `.env` 文件
